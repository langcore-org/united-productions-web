# 4シーン説明型インフォグラフィック プロンプトテンプレート集

**対応モデル**: Gemini 3 Pro Image Preview (gemini-3-pro-image-preview)
**フォーマット**: 16:9 横長、4シーン構成
**スタイル**: プロフェッショナルな説明型インフォグラフィック・日本語テキスト
**特徴**: 4つのシーンで段階的に情報を説明、クリーンなベクターイラスト

## 基本テンプレート（4シーン説明型）

### TV番組企画向けテンプレート
```
Create a 4-scene infographic explaining TV show concept [番組名]:

SCENE 1 - 企画紹介:
Header: "SCENE 1 - 企画紹介" with blue background
Title: 新企画「[番組名]」始動！
Visual: 番組コンセプトを表すメインビジュアル
Text: 番組の概要、ターゲット視聴者、コンセプト説明
Icons: 賞金、参加者数などの重要数値

SCENE 2 - ルール・方法:
Header: "SCENE 2 - ルール説明" with blue background
Title: [挑戦内容]の詳細
Visual: 参加者やルールを示すイラスト
Text: 具体的なルール、参加方法、制限事項
Elements: ステップバイステップの視覚化

SCENE 3 - クライマックス:
Header: "SCENE 3 - クライマックス" with blue background
Title: 最大の見せ場！
Visual: 番組のハイライトシーンの視覚化
Text: 最も盛り上がるポイント、視聴者の注目点
Effects: ドラマチックな演出効果

SCENE 4 - 結果発表:
Header: "SCENE 4 - 結果発表" with blue background
Title: 勝者、決定の瞬間
Visual: 優勝者表彰、賞金授与のシーン
Text: 結果の決定方法、審査基準、次のステップ
Success: 賞金や賞品の視覚化

Visual Style:
- Clean vector illustration style
- Rounded corner boxes for each scene
- Pastel color palette (light blue, soft pink, yellow)
- Simple character illustrations
- Professional infographic aesthetic
- All text in Japanese
```

## スタイル要素

### ヘッダーバー
- 背景色: ブルー (#2196F3 - #1976D2)
- テキスト色: ホワイト
- フォント: 太字、明瞭
- 高さ: シーンボックスの約10-15%

### シーンボックス
- 角丸: 中程度の丸み
- ボーダー: 細い線または影効果
- 背景: 白またはごく薄い色
- パディング: 適切な余白

### イラストスタイル
- ベクターイラスト: クリーンなライン
- キャラクター: シンプルで親しみやすい
- アイコン: フラットデザイン
- カラー: パステルトーン

### テキスト配置
- タイトル: 大きく明瞭
- 本文: 読みやすいサイズ
- 箇条書き: 視覚的に整理
- 数値: 強調表示

## 使用例

### 例: TV番組企画「神業アーチェリーチャレンジ」
```
Create a 4-scene infographic about archery balloon challenge TV show:

SCENE 1 - 企画紹介:
Header: "SCENE 1 - 企画紹介" (blue background)
Title: 新企画「神業アーチェリーチャレンジ」始動！
Visual: 風船とアーチェリーの矢のイラスト
Text: 空中に放たれた一つの風船を、矢で射貫く。究極の集中力、技術、そして運が試される前代未聞の挑戦から、誰を勝ち抜くことができるのか？
Icons: 優勝賞金100万円の札束とトロフィー
Bottom text: 5人のトップアスリートが集結

SCENE 2 - 挑戦開始:
Header: "SCENE 2 - 挑戦開始" (blue background)
Title: ターゲット: 空中を舞う一つの風船
Visual: 5人のアーチェリー選手が弓を構えるイラスト
Text: 風に乗り、予測不能な動きで浮遊する風船。そのわずかな的を狙って矢を放つ。選手たちの精神と技術が試される。
Elements: 弓を構える選手たち、矢の番号「1」、空中の風船
Bottom text: チャンスは一度きり

SCENE 3 - クライマックス:
Header: "SCENE 3 - クライマックス" (blue background)
Title: 矢、放たれる！
Top visual: 矢が空中を飛ぶスローモーション演出、5本の矢が様々な色で光跡を描く
Text: 風船を破り、選手たちが放った矢の軌跡がスリング効果で光を放ちながら空に描く。
Bottom visual: 風船がわずかにゆがみながら、矢が命中する瞬間のスローモーション
Text: 風船をわずかに歪ませる極い一瞬、風船に向かって一直線に飛ぶ矢をスローモーションで映し出し、緊張感を最大化する。
Emphasis: 劇的瞬間！風船、貫通！

SCENE 4 - 結果発表:
Header: "SCENE 4 - 結果発表" (blue background)
Title: 勝者、決定の瞬間
Visual: 表彰台に立つ優勝者、主催者が賞金ボードを渡す
Text: 見事、風船を射抜いた選手が勝者に。タイムが早かった場合は、タイムや判定により順位が決定される。複数の成功者が出た場合は、タイムや判定により1位を決定。
Success visual: ¥1,000,000 賞金ボード
Bottom text: 100万円の授与式

Visual Style:
- Clean vector illustrations
- Rounded corners on scene boxes
- Light blue, soft pink, yellow accents
- Simple character designs
- Professional TV show aesthetic
- All Japanese text with clear fonts
```
