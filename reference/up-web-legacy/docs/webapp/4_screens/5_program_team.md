# 5. Program/Team - 番組・チーム管理画面設計

## 概要

番組とチームの作成・管理画面。番組はGoogle Driveのルートディレクトリと連携し、チームはエージェントタイプと参照ファイルを設定する。

## Google Drive フォルダ設定の階層構造

```
Workspace (ワークスペース設定: /settings/drive)
  └─ Service Account JSON のみ設定
     ※ フォルダIDはここでは設定しない

Program (番組設定: PG-002, PG-004)
  └─ 参照フォルダ (rootFolderId)
     - @メンションで参照できるルートフォルダ
     - 番組作成時に設定（STEP 2: Google Drive連携）

Team (チーム設定: TM-001, TM-002)
  └─ 出力フォルダ (outputFolderId)
     - 成果物を保存するフォルダ
     - チーム作成時に設定（STEP 5: 出力先設定）
```

**設計理由:**
- **参照フォルダ**: 番組ごとに異なる資料フォルダを使用するため、番組単位で設定
- **出力フォルダ**: チームごとに成果物の種類が異なるため、チーム単位で設定
- **Service Account**: ワークスペース全体で共有する認証情報のため、ワークスペース単位で設定

## 画面一覧

| 画面ID | 画面名 | パス | 説明 |
|--------|--------|------|------|
| PG-001 | 番組一覧 | `/[slug]/programs` | ワークスペース内の番組一覧 |
| PG-002 | 番組作成 | `/[slug]/programs/new` | 新規番組作成 |
| PG-003 | 番組詳細 | `/[slug]/programs/[programId]` | 番組のチーム一覧 |
| PG-004 | 番組設定 | `/[slug]/programs/[programId]/settings` | 番組設定編集 |
| TM-001 | チーム作成 | `/[slug]/programs/[programId]/teams/new` | 新規チーム作成 |
| TM-002 | チーム設定 | `/[slug]/programs/[programId]/teams/[teamId]/settings` | チーム設定編集 |

## ユーザーシナリオ

### シナリオ 1: 新しい番組の作成

```mermaid
sequenceDiagram
    actor User as ディレクター
    participant Programs as 番組一覧
    participant New as 番組作成画面
    participant Picker as フォルダピッカー
    participant Detail as 番組詳細

    User->>Programs: 「+ 新しい番組」クリック
    Programs->>New: 番組作成画面へ

    User->>New: 基本情報入力
    Note over User,New: 名前、説明、期間

    User->>New: 「フォルダを選択」クリック
    New->>Picker: フォルダピッカー表示
    User->>Picker: ルートフォルダ選択
    Picker->>New: 選択完了

    User->>New: 「番組を作成」クリック
    New->>Detail: 番組詳細（チーム一覧）へ遷移
    Note over Detail: チームが0件の状態で表示
```

### シナリオ 2: 番組にチームを追加

```mermaid
sequenceDiagram
    actor User as AD
    participant Detail as 番組詳細

    participant Picker as ファイルピッカー
    participant Chat as チャット画面

    User->>Detail: 「+ 新しいチーム」クリック
    Detail->>NewTeam: チーム作成画面へ

    User->>NewTeam: 基本情報入力
    User->>NewTeam: エージェントタイプ選択（リサーチ）

    User->>NewTeam: 「ファイル/フォルダを追加」クリック
    NewTeam->>Picker: ファイルピッカー表示
    User->>Picker: 参照するファイルを選択
    Picker->>NewTeam: 追加完了

    User->>NewTeam: 出力先フォルダ選択
    User->>NewTeam: 「チームを作成」クリック
    NewTeam->>Chat: チャット画面へ遷移
    Note over Chat: 新しいセッション開始可能
```

### シナリオ 3: チームの参照ファイル更新

```mermaid
sequenceDiagram
    actor User as AD
```
