# 📊 最終調査報告書
## 「QuizKnock いざわたくし vs 天才小学生 究極対決SP」番組企画

---

**作成日**: 2025年1月1日
**調査期間**: 2024年12月～2025年1月
**対象**: テレビ特番企画提案
**予算規模**: 500万円

---

## 🎯 エグゼクティブサマリー

### 企画概要
QuizKnockのCEO伊沢拓司氏が、各分野で突出した才能を持つ小学生たちと専門性の高いクイズ対決を繰り広げる50分間の特番企画。単純な知識問題ではなく、**思考力・論理力・創造力**を競う新感覚バトルコンテンツ。

### 核心的価値提案
- **本格性**: 台本なしの真剣勝負による緊張感とリアリティ
- **教育性**: STEAM教育トレンドに合致した学習促進効果
- **エンタメ性**: 世代間対決の構図による幅広い視聴者訴求
- **社会性**: 才能発掘・支援への社会的関心喚起

### 推奨判断
本企画は**実行推奨**。教育コンテンツへの社会的ニーズ、QuizKnockブランドの認知度、出演候補者の質の高さを総合的に勘案し、視聴率7-10%、高いSNS拡散効果が期待できる。

---

## 📺 番組構成・演出設計

### 全体構成（50分）
1. **オープニング（5分）**: 天才小学生紹介VTR、対決宣言
2. **数学バトル（15分）**: 日本科学未来館での論理思考対決
3. **将棋的思考バトル（15分）**: 東京将棋会館での推理・戦略クイズ
4. **テクノロジー対決（10分）**: プログラミング体験施設での実装力勝負
5. **結果発表・感想戦（5分）**: 総合勝敗と出演者コメント

### 演出の特徴
- **ロケーション重視**: 各専門分野に適した格調高い撮影地
- **リアルタイム性**: 生の思考プロセスを可視化
- **教育的配慮**: 勝敗よりも思考過程に焦点
- **世代間交流**: 互いから学び合う姿勢を演出

## 🏢 ロケーション戦略

### 選定基準
1. **専門性**: 各分野の権威ある施設
2. **撮影適性**: 照明・音響・画角の良さ
3. **アクセス**: 都内移動効率
4. **コスト**: 予算内での利用可能性

### 推奨ロケ地

#### 🔬 日本科学未来館（数学バトル会場）
- **選定理由**: 科学的権威性、未来的ビジュアル
- **撮影ポイント**: ガラス張り研究室、巨大地球ディスプレイ
- **コスト**: 40万円（設備使用料込み）
- **代替案**: 理化学研究所（より専門的だが許可取得困難）

#### 🏆 東京将棋会館（論理思考バトル会場）
- **選定理由**: 格式ある和の空間、集中力演出に最適
- **撮影ポイント**: 対局室の緊張感、畳・木材の質感
- **コスト**: 30万円（新会館での撮影効果）
- **注意点**: 土日の大会開催日は避ける要

#### 💾 プログラミング体験施設（テクノロジー対決会場）
- **候補**: Life is Tech!スクール、プログラミング教室
- **撮影ポイント**: 子どもが実際にコーディングする姿
- **コスト**: 30万円（機材・設営費込み）
- **付加価値**: 教育機関との連携可能性

## 🚨 リスク分析・対策

### 高リスク項目

#### 児童出演関連リスク
- **法的リスク**: 児童福祉法、労働基準法遵守必須
- **対策**: 撮影時間制限、休憩確保、保護者常時同伴
- **追加考慮**: プライバシー保護、学習への影響最小化

#### 撮影・運営リスク
- **ロケ地確保**: 人気施設の予約競争
- **対策**: 早期予約、代替案複数準備
- **天候リスク**: 屋内撮影メインのため影響軽微

### 中リスク項目

#### 予算オーバーリスク
- **要因**: 延長撮影、機材トラブル、追加演出
- **対策**: 予備費確保、詳細スケジュール管理

#### 出演者体調管理
- **要因**: 小学生の急な体調不良
- **対策**: 予備日程確保、代替出演者リスト準備
