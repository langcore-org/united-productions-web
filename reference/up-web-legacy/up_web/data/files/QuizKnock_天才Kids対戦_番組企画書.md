# QuizKnock × 天才Kids 対戦番組企画書

## 番組概要

**番組タイトル**: 「QuizKnock VS 天才Kids ～究極の頭脳バトル～」
**放送時間**: 50分
**予算**: 500万円
**企画意図**: 各分野のスペシャリスト小学生とQuizKnockメンバーがリアルなフィールドで真剣勝負

## 出演者

### QuizKnock側
- **伊沢拓司** (メインホスト・全分野対応)
- **東問・東言** (算数・将棋担当)
- **ふくらP** (ロボット・プログラミング担当)
- **須貝駿貴** (科学・工学担当)

### 天才Kids側 (2025年実績者)
- **土田悠真君** (算数・将棋の天才/新潟県小6)
- **小嶋奏次郎さん** (ロボット工学の天才/大阪府小4)
- **プログラミングコンテスト優勝者** (Tech Kids Grand Prix 2025)
- **算数オリンピック金メダリスト**

## 番組構成 (50分)

### オープニング (5分)
- 番組コンセプト紹介
- 出演者プロフィール紹介
- 各バトルフィールド紹介

### 第1ラウンド: 算数バトル (10分)
**ロケ地**: 理化学研究所（埼玉・和光市）
**協力**: 公文教育研究会
- 制限時間内の計算競技
- 数学的思考問題での対戦
- 研究施設での実践的問題解決

### 第2ラウンド: ロボット対決 (10分)
**ロケ地**: トヨタ産業技術記念館（愛知・名古屋）
**協力**: トヨタ自動車
- 与えられた課題でロボット組立競争
- プログラミング実演バトル
- 実際の製造ライン見学とクイズ

### 第3ラウンド: プログラミング勝負 (10分)
**ロケ地**: サイバーエージェント本社（東京・渋谷）
**協力**: サイバーエージェント
- リアルタイムコーディング対戦
- アプリケーション開発競争
- IT企業の最前線での実践バトル

### 第4ラウンド: 将棋真剣勝負 (10分)
**ロケ地**: 日本将棋会館（東京・千駄ヶ谷）
**協力**: 日本将棋連盟
- プロ棋士立会いでの公式戦
- 制限時間での真剣勝負
- 将棋文化と歴史の紹介

### エンディング・総合結果 (5分)
- 各ラウンド結果発表
- 参加者インタビュー
- 次回予告・メッセージ
