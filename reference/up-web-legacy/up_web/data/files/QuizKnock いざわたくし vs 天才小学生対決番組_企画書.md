# QuizKnock いざわたくし vs 天才小学生対決番組 企画書

## エグゼクティブサマリー
QuizKnockの人気クリエイター「いざわたくし」と、2025年時点の各分野天才小学生によるガチンコクイズ対決番組。小学生の得意分野に特化したロケーション撮影を行い、教育とエンタメの両面を追求する50分尺の特番企画。

---

## 主要なインサイト

### Insight 1: 2025年の天才小学生市場は空前の盛り上がり

**発見の背景**:
- 算数オリンピック参加者が6,000人超（過去最高記録）
- 藤井聡太効果で将棋人気が爆発的拡大、9歳プロ棋士も誕生
- CBCテレビ「新天才クイズ2025冬」など既存番組も進化中

**裏付けデータ**:
- 算数オリンピック2024年：参加者6,000人以上（史上最高）
- 将棋界：世界最年少9歳プロ棋士・藤田怜央初段が誕生
- 音楽コンクール：全日本学生音楽コンクール（毎コン）4年生以上参加可能

**参考リソース**:
- 算数オリンピック公式サイト (Score: 9/10)
- 全日本学生音楽コンクール (Score: 8/10)

**構成案**:
オープニングで各分野の天才小学生の実力をダイジェストで紹介。「今、小学生の才能がヤバい！」というトーンで視聴者の興味を引く

### Insight 2: 分野別天才小学生の特徴と対決可能性

**発見の背景**:
現在の天才小学生は従来の「勉強ができる子」を超えて、特化型の才能を持つ子が急増中

**分野別特徴**:
- **STEM系**: 理科実験コンクール、科学オリンピック参加者多数
- **音楽系**: ピアノ・バイオリンコンクールが活況
- **スポーツ系**: 水泳52.6%、体操35%が人気、早期専門化より多競技推奨
- **頭脳ゲーム系**: 将棋・囲碁人口が爆発的増加

**参考リソース**:
- 自然科学観察コンクール（シゼコン） (Score: 8/10)
- 全日本ジュニアクラシック音楽コンクール (Score: 7/10)

**構成案**:
各分野から1-2名選出し、その分野の専門機関でクイズ対決。いざわさんの博識 vs 小学生の専門特化という構図

### Insight 3: ロケ地として最適な専門機関が充実

**発見の背景**:
東京を中心に、各分野の専門機関が小学生向けプログラムを積極展開中

**主要ロケ地候補**:

#### STEM・科学系
- **科学技術館**（千代田区）: ロケ撮影対応明記、体験型展示豊富
- **日本科学未来館**（お台場）: 最新技術展示、ロボット・AI体験可能
- **多摩六都科学館**（西東京市）: 宇宙体験、月重力体験装置

#### 頭脳ゲーム系
- **新・将棋会館**（渋谷→千駄ヶ谷）: 2024年9月新築、放送設備完備
- **関西将棋会館**（大阪→高槻）: 2024年秋移転予定、モダン設備

#### 音楽系
- **浜離宮朝日ホール**: 高品質アコースティック
- **東京文化会館**: 大・小ホール完備、権威性高い
- **ガルバホール**（新宿）: ベーゼンドルファーピアノ、撮影向き74席

#### IT・プログラミング系
- **TEPIA先端技術館**（外苑前）: ロボット・プログラミング体験
- **N Code Labo**（角川ドワンゴ学園）: 本格的プログラミング教室
- **JSOL**（麹町）: 企業によるプログラミング教育支援

**参考リソース**:
- 科学技術館 (Score: 9/10)
- 浜離宮朝日ホール (Score: 8/10)

**構成案**:
各ロケ地で「その道のプロ」が出題・解説役として登場。施設見学も兼ねた教育的要素を盛り込む

### Insight 4: エンタメ性と教育性の絶妙なバランス

**発見の背景**:
視聴者は「天才小学生すごい！」だけでなく、「自分も学べる」要素を求めている

**番組設計のポイント**:
- いざわさんの「なぜその答えに？」という素朴な疑問が視聴者代表
- 小学生の思考プロセスを丁寧に描くことで学習番組的価値も創出
- 勝負の行方より「どちらも素晴らしい才能」という着地

**想定構成** (50分):
- オープニング・出演者紹介: 5分
- 第1ラウンド（科学系）: 12分
- 第2ラウンド（音楽系）: 12分
- 第3ラウンド（頭脳ゲーム系）: 12分
- 結果発表・感想戦: 9分

**参考リソース**:
- CBCテレビ新天才クイズ (Score: 7/10)

**構成案**:
各ラウンド後に「いざわ解説タイム」を設け、大人の視点から学びを深める

### Insight 5: 予算500万円での実現可能性

**発見の背景**:
ロケ地の多くは教育目的での使用に協力的、コストを抑えた制作が可能

**概算予算配分**:
- 出演料（いざわ・小学生・専門家）: 150万円
- ロケ地使用料・交通費: 100万円
- クルー費（1日×3ロケ）: 180万円
- 編集・MA: 50万円
- その他諸経費: 20万円

**コスト削減のポイント**:
- 科学館等は教育目的で使用料減免の可能性
- 小学生出演料は家族も含めた記念品・体験重視
- いざわさんのYouTube人気でWin-Win関係構築

**参考リソース**:
一般的なロケ番組制作費を参考に算出

**構成案**:
予算制約を逆手に取り、「リアルな学びの場」での撮影により教育的価値を強調

## 制作スケジュール案

### Pre-Production（企画決定後1ヶ月）
- 天才小学生キャスティング（算数・音楽・将棋各1名）
- ロケ地使用許可取得
- 問題作成・監修依頼

### Production（撮影3日間）
- Day1: 科学技術館（STEM系対決）
- Day2: 音楽ホール（音楽系対決）
- Day3: 将棋会館（頭脳ゲーム系対決）

### Post-Production（1ヶ月）
- 編集・MA・グラフィック制作
- 専門家解説VTR制作

## 期待効果

### 視聴率・話題性
- QuizKnockファン × 教育関心層の幅広い視聴者獲得
- SNSでの拡散期待（いざわさんの素直な驚きとリアクション）
- 続編・レギュラー化の可能性

### 教育的インパクト
- 小学生の学習意欲向上
- 専門分野への興味喚起
- 親子での視聴促進

### 制作サイド
- 教育番組としての社会的意義
- ロケ地施設との良好な関係構築
- 新しいクイズ番組フォーマットの確立

**この企画は、エンタメ性と教育性を両立し、予算内で実現可能な魅力的なコンテンツです。天才小学生たちの輝く才能と、いざわさんの人間味あふれるリアクションが、視聴者に感動と学びを届けます。**
