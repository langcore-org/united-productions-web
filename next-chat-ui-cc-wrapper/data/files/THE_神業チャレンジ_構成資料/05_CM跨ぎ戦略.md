
# インフォグラフィック仕様: CM跨ぎ戦略

## 概要
視聴者の離脱を防ぎ、CM後の継続視聴を促進するための戦略的なビジュアル

## CM跨ぎポイント設計

### Scene 1: CM Break Point ① (約15:00付近)
**タイミング**: 本編パート1のクライマックス直前
**内容**:
- サスペンス指標、ストップウォッチ、クエスチョンマークアイコン
- テキスト: 「ここからが本番！」
- ビジュアル: 高緊張ドラマ的美学
- バックグラウンドミュージック: 盛り上がり系効果音
- ターゲット: 「続きが見たい」という心理を喚起

### Scene 2: いざわの反応テーザー
**内容**:
- いざわの驚嘆/決意顔、アクションポーズ
- テキスト: 「この後、いざわが昆虫採集に初挑戦！」
- ビジュアル: 冒険/プレビュー興奮感
- 背景: 冒険テーマ、フォレストグリーン
- ターゲット: 「何が起こるのか」という期待感

### Scene 3: CM Break Point ② (約35:00付近)
**タイミング**: 実地体験セグメント終了時
**内容**:
- 大きな達成モーメント、トロフィー/メダルイメージリー
- テキスト: 「天才Kids の驚きの正解が...」
- ビジュアル: クリフハンガー的な劇的展開
- 背景: ドラマティック、謎めいた紫系
- ターゲット: 「答えが知りたい」という知識欲を刺激

### Scene 4: 復帰テーザー
**内容**:
- 教授キャラクター、説明グラフィックス
- テキスト: 「専門家による白熱解説はCM明けで！」
- ビジュアル: 教育的期待感
- バックグラウンドミュージック: リターン予告系効果音
- ターゲット: CM明けの視聴継続

## デザイン方針
- ドラマティック、サスペンスフル
- 色使い: 赤（ドラマ）、黄色（興奮）、青（期待）
- テキスト: すべて日本語、ボールド系フォントで影響力
- ビジュアル: クイックカット、効果音との同期

## 視聴率維持戦略
1. **CM前煽り**: 「この後、衝撃！」という明確なクリフハンガー
2. **プレビュー**: 「見逃すな」という心理的プレッシャー
3. **専門家解説**: 「知識が得られる」という学習欲刺激
4. **タイムマネジメント**: 各Break pointで正確に心理的ターニングポイントを設定
