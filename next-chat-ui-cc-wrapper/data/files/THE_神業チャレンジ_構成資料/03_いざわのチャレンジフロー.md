
# インフォグラフィック仕様: いざわのチャレンジフロー

## 概要
QuizKnockのいざわたくしが、天才Kidsに挑むプロセスを4ステップで可視化

## 進行フロー

### Scene 1: クイズバトル Round 1（初級～中級）
**内容**:
- クイズカード、クエスチョンマークアイコン
- テキスト: 「初級～中級クイズで対決」
- ビジュアル: いざわ vs Kidsのアイコン対比
- 背景: クイズ番組的な紫・金色美学
- ターゲット: 子どもに「いざわが答えられるかな？」というドキドキ感

### Scene 2: 実地チャレンジ体験
**内容**:
- 化石発掘、昆虫採集、数学パズル解きシーン
- テキスト: 「実地でのチャレンジ体験」
- ビジュアル: ダイナミックなアクションポーズ、屋外冒険テーマ
- 背景: 自然/冒険テーマ
- ターゲット: 子どもに「自分もやってみたい」という参加欲

### Scene 3: クイズバトル Round 2（上級）
**内容**:
- 高難易度クイズ表示（★★★）、トロフィーアイコン
- テキスト: 「上級クイズで実力発揮！」
- ビジュアル: 激戦的な対決シーン
- 背景: 激戦/高緊張美学
- ターゲット: 視聴者の「どちらが勝つのか」という興奮

### Scene 4: 専門家解説と成長
**内容**:
- いざわの驚き表情、教授/専門家キャラクター
- テキスト: 「プロの解説で理解を深める」
- ビジュアル: 学習進捗インジケーター、成長グラフ
- 背景: 教育パネルディスカッション
- ターゲット: 親への「学習効果」の訴求

## デザイン方針
- ダイナミック、進行型、成長を表現
- 色使い: 紫（知識）、金（達成）、緑（成長）
- 感情表現: 😲（驚き）、💪（努力）、🎉（達成）
- テキスト: すべて日本語

## 視聴者反応設計
- 子ども：アクション、驚き、エンタメ性
- 親：教育的価値、成長過程、知識習得
