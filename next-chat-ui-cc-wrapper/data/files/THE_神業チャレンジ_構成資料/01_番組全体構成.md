
# インフォグラフィック仕様: 番組全体構成

## 概要
『THE 神業チャレンジ × QuizKnock天才Kids対決』の全体フローを4シーンで可視化

## 4シーン構成

### Scene 1: Opening (0:00-5:00)
**内容**: 番組冒頭の掴みシーン
- Chocolate Planet MCがスタジオで今日のテーマを紹介
- タイトル: 「神業 vs 天才Kids 対決！」をスパークル効果付きで表示
- ビジュアル: 明るいスタジオセット、視聴者を引き込む導入画面
- ターゲット: C・T層の「面白そう」という期待感を醸成

### Scene 2: Main Challenge Part 1 (5:00-35:00)
**内容**: 複数の天才Kidsによるチャレンジシーン
- 昆虫博士、恐竜博士、数学天才などが画面に登場
- タイトル: 「本気の挑戦」と劇的効果音
- ビジュアル: スプリットスクリーン、各分野アイコン表示
- ターゲット: Kidsへの「すごい！」という驚き、親への「子どもの才能発掘」の問い

### Scene 3: CM Break Strategic Point (35:00付近)
**内容**: 視聴者を次パートへ引き込むクリフハンガー
- チャレンジ結果の大きな盛り上がりシーン
- テキスト: 「この後、衝撃の展開が！」
- ビジュアル: サスペンスフルなカウントダウン、クエスチョンマーク
- ターゲット: 視聴者の離脱を防止、CM後の視聴継続を促進

### Scene 4: Closing & Parent Message (50:00-58:00)
**内容**: 締めくくりと親向けメッセージ
- MCが今回のレッスンを振り返り
- タイトル: 「子どもの才能を見つけるコツ」
- ビジュアル: 温かみのある家族画像、メッセージボード形式のヒント表示
- ターゲット: 親世代への行動喚起（「うちの子にもできることがあるのでは？」）

## デザイン方針
- 明るく、遊び心のある日本のバラエティ番組的美学
- 色使い: 青（信頼・知識）、黄色（知性・興奮）、オレンジ（エネルギー・親しみ）
- テキスト: すべて日本語、フォントサイズ最小24pt（ヘッダー）
- 装飾: 日本語絵文字・装飾要素（✨🌟💡）を活用

## 制作工程
1. イラストレーターによる基礎画面設計
2. テキスト・カラーリング調整
3. 番組用グラフィック統合
4. 放送品質での出力（解像度1920x1080以上）
