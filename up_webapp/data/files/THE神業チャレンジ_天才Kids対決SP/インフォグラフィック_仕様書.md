# インフォグラフィック仕様書
## THE 神業チャレンジ 2時間SP「天才Kids対決」

---

## インフォグラフィック目的

各主要企画セグメントについて、複雑な概念・流れを**視覚的に可視化**し、以下を実現する：

1. **プロモーション効果**: SNS・番組予告で「この番組は何をするのか」を一瞬で理解させる
2. **編集効率化**: スタジオVTR内に差し込むことで、プレゼンテーション時間を短縮
3. **視聴者理解**: 親世代が「才能教育とは何か」を視覚的に学べる教材化

---

## デジタル資産管理

### ファイル命名規則
```
THE_神業チャレンジ_[グラフィック番号]_[タイトル]_[用途]_v[バージョン].png
例：THE_神業チャレンジ_03_知識クイズバトル_スタジオVTR_v1.png
```

### 納品形式
- **静止画**: PNG 1920×1080 / 1080×1080（透明背景対応）
- **動画**: MP4 H.264 1920×1080 30fps（テロップ・トランジション付き）
- **編集データ**: Adobe Illustrator AI / AfterEffects AEP

### バージョン管理
- 初稿：v1
- 修正版：v1.1、v1.2...
- 最終版：vFinal
