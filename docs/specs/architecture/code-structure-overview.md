## 2. 認証フロー

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   ユーザー    │────▶│  /auth/signin    │────▶│  Google OAuth   │
└─────────────┘     └──────────────────┘     └─────────────────┘
                                                          │
                                                          ▼
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  アプリ画面   │◀────│  /api/auth/...   │◀────│  認証コールバック  │
└─────────────┘     └──────────────────┘     └─────────────────┘
                            │
                            ▼
                    ┌──────────────────┐
                    │  lib/auth-options │
                    │  - JWT戦略        │
                    │  - セッション管理  │
                    └──────────────────┘
                            │
                            ▼
                    ┌──────────────────┐
                    │  lib/prisma.ts   │
                    │  - ユーザー保存   │
                    └──────────────────┘
```

### 主要ファイル

| ファイル | 役割 |
|---------|------|
| `app/api/auth/[...nextauth]/route.ts` | NextAuth.js APIエンドポイント |
| `lib/auth-options.ts` | 認証設定（Google OAuth、JWT戦略） |
| `middleware.ts` | 認証チェックミドルウェア |
| `app/(authenticated)/layout.tsx` | 認証必須レイアウト |

## 4. Agent（機能別チャット）システム

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         lib/chat/agents.ts                              │
│                                                                         │
│  Agent = 特定の用途に特化したチャット機能（AI Agent）                      │
│                                                                         │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐    │
│  │   general   │  │research-cast│  │research-loc │  │research-info│    │
│  │  （一般）    │  │（出演者）    │  │ （場所）     │  │ （情報）     │    │
│  └─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘    │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐    │
│  │   minutes   │  │  proposal   │  │  na-script  │  │   ...       │    │
│  │ （議事録）   │  │ （企画立案） │  │ （NA原稿）  │  │             │    │
│  └─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘    │
│                                                                         │
│  各Agentは以下を定義:                                                    │
│  - systemPrompt: システムプロンプト                                       │
│  - placeholder: 入力欄のプレースホルダー                                   │
│  - outputFormat: markdown/plaintext                                       │
│  - toolOptions: Web検索、X検索等の有効/無効                               │
│  - promptKey: DB管理プロンプトとの紐付けキー                               │
│                                                                         │
│  注: lib/chat/gems.ts は後方互換性のため維持（エイリアスのみ）              │
└─────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                      lib/prompts/db.ts                                  │
│                                                                         │
│  DB管理プロンプト（Single Source of Truth）                               │
│                                                                         │
│  - DEFAULT_PROMPTS: フォールバック用デフォルトプロンプト                   │
│  - getPromptFromDB(): DBからプロンプトを取得                              │
│  - updatePromptWithVersion(): バージョン管理付き更新                      │
│                                                                         │
│  プロンプトキー:                                                          │
│  GENERAL_CHAT, RESEARCH_CAST, RESEARCH_LOCATION, RESEARCH_INFO,          │
│  RESEARCH_EVIDENCE, MINUTES, PROPOSAL, TRANSCRIPT                        │
└─────────────────────────────────────────────────────────────────────────┘
```

## 7. 主要ファイル一覧と役割

### 7.1 設定・基盤ファイル

| ファイル | 役割 | 依存先 |
|---------|------|--------|
| `package.json` | 依存パッケージ定義 | - |
| `tsconfig.json` | TypeScript設定 | - |
| `next.config.ts` | Next.js設定 | - |
| `tailwind.config.ts` | Tailwind CSS設定 | - |
| `middleware.ts` | 認証ミドルウェア | `lib/auth-options.ts` |
| `.env.local` | 環境変数 | - |

### 7.2 認証関連

| ファイル | 役割 | 依存先 |
|---------|------|--------|
| `lib/auth-options.ts` | NextAuth.js設定 | `lib/prisma.ts` |
| `app/api/auth/[...nextauth]/route.ts` | 認証APIエンドポイント | `lib/auth-options.ts` |
| `types/next-auth.d.ts` | セッション型拡張 | - |

### 7.3 データベース

| ファイル | 役割 | 依存先 |
|---------|------|--------|
| `prisma/schema.prisma` | DBスキーマ定義 | - |
| `lib/prisma.ts` | Prisma Clientインスタンス | `@prisma/client` |

### 7.4 LLM統合

| ファイル | 役割 | 依存先 |
|---------|------|--------|
| `lib/llm/types.ts` | LLM関連の型定義 | - |
| `lib/llm/config.ts` | プロバイダー設定 | `lib/llm/types.ts` |
| `lib/llm/factory.ts` | LLMクライアント生成 | `lib/llm/langchain/adapter.ts` |
