# 議事録作成

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

**文字起こしテキストから、きれいな議事録を自動作成**します。決定事項やTODOを自動で抽出して、読みやすい形式に整理します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **Zoom会議の議事録を作りたい** | Zoomの文字起こしを貼り付ける |
| **打ち合わせの内容を整理** | 録音から文字起こししたテキスト |
| **TODOリストを抽出** | 「この会議のTODOをリストアップして」 |
| **決定事項を確認** | 「決定事項だけを抽出して」 |

---

## 入力と出力の例

### 例：打ち合わせ会議の議事録作成

**あなたの入力:**
```
会議日：2026年2月20日
参加者：山田、佐藤、鈴木、田中

山田：では、新企画の進捗確認を始めます。
佐藤さん、デザインはどうですか？

佐藤：はい、トップ画面は80%完了しました。
残りはアイコンの選定です。

山田：いつまでに完了しますか？
佐藤：来週水曜日までに提出します。

鈴木：開発はAPI連携が完了し、次はテスト工程です。
山田：テスト期間はどのくらい？
鈴木：2週間で完了予定です。

田中：次回は3月6日に進捗確認をしましょう。
```

**AIの出力:**

━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━

■ デザイン進捗（佐藤）

## 画面の使い方

1. **文字起こしテキスト**を入力欄に貼り付ける
2. **送信ボタン**をクリック
3. AIが**整形された議事録**を作成
4. **コピー**して文書として使用

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **会議情報** | 日時、参加者、議題を自動抽出 |
| **議事内容** | 話題ごとに整理された内容 |
| **決定事項** | ✅ マーク付きで明確に抽出 |
| **TODOリスト** | 担当者・期限付きで整理 |
| **次回予定** | 次回会議の情報 |

---

## 入力のコツ

| 入力タイプ | 説明 |
|-----------|------|
| **Zoom文字起こし** | Zoomの文字起こし機能で出力されたテキストをそのまま貼り付け |
| **Premiere Pro書き起こし** | 動画編集ソフトの書き起こし結果 |
| **音声認識アプリ** | Otter.ai、Nottaなどの出力 |
| **手動メモ** | 会議中にメモした内容 |
