# 議事録作成

> **最終更新**: 2026-02-25 14:10

---

## この機能でできること

**Zoomの文字起こし（VTTまたはTXTファイル）から、きれいな議事録を自動作成**します。決定事項やTODOを自動で抽出して、読みやすい形式に整理します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **Zoom会議の議事録を作りたい** | Zoomの文字起こしファイルをそのまま貼り付ける |
| **打ち合わせの内容を整理** | 録音から文字起こししたテキスト |
| **TODOリストを抽出** | 「この会議のTODOをリストアップして」 |
| **決定事項を確認** | 「決定事項だけを抽出して」 |

---

## Zoom文字起こしの形式

Zoomでは**2種類の形式**で文字起こしを取得できます。

### 形式1：VTTファイル（クラウドレコーディング）

クラウドレコーディングと一緒に生成される字幕ファイルです。

```
WEBVTT

1
00:00:05.000 --> 00:00:10.000
山田: では、新企画の進捗確認を始めます。

2
00:00:10.000 --> 00:00:15.000
山田: 佐藤さん、デザインはどうですか？
```

| 特徴 | 説明 |
|------|------|
| **ファイル形式** | `.vtt` |
| **取得方法** | Zoom Webポータルからダウンロード |
| **生成時間** | 会議終了後30分程度 |
| **含まれる情報** | タイムスタンプ、話者名、発言内容 |

---

### 形式2：TXTファイル（フルトランスクリプト）

会議中のフルトランスクリプトを保存したテキストファイルです。

```
14:02:05  山田: では、新企画の進捗確認を始めます。
14:02:10  山田: 佐藤さん、デザインはどうですか？
14:02:15  佐藤: はい、トップ画面は80%完了しました。
14:02:20  佐藤: 残りはアイコンの選定です。
```

| 特徴 | 説明 |
|------|------|
| **ファイル形式** | `.txt` |
| **取得方法** | 会議中に「文字起こしを保存」ボタンをクリック |
| **生成タイミング** | 会議中リアルタイムで保存可能 |
| **含まれる情報** | 時刻、話者名、発言内容 |

---

## どちらの形式を使うか

| 形式 | おすすめ度 | 使い分け |
|------|-----------|---------|
| **VTT** | ⭐⭐⭐ | クラウド録画後に議事録を作成する場合。タイムスタンプが正確。 |
| **TXT** | ⭐⭐⭐ | 会議中または直後にすぐ保存したい場合。手軽に取得できる。 |

**どちらでも同じように議事録を作成できます。**

## 画面の使い方

1. **Zoomの文字起こしファイル**（VTTまたはTXT）を開いてコピー
2. 入力欄にそのまま貼り付ける
3. **送信ボタン**をクリック
4. AIが**整形された議事録**を作成
5. **コピー**して文書として使用

---

## Zoom文字起こしの取得方法

### 方法1：VTTファイルを取得（クラウドレコーディング）

**前提条件**
- Zoomの**Proプラン以上**
- **クラウドレコーディング**を有効にしている

**手順**

1. **会議をクラウド録画する**
   - 会議中に「録画」→「クラウドに録画」を選択

2. **会議終了後、文字起こしを待つ**
   - 生成には**30分程度**かかります

3. **Zoom Webポータルでダウンロード**
   1. [Zoom Webポータル](https://zoom.us/signin)にログイン
   2. 「**レコーディング**」→「**クラウドレコーディング**」を選択
   3. 対象の会議をクリック
   4. 「**オーディオ文字起こし**」の「**VTT**」をクリックしてダウンロード

4. **VTTファイルを開いてコピー**
   - テキストエディタで開く
   - 全選択（Ctrl+A）→コピー（Ctrl+C）
   - Teddyに貼り付け（Ctrl+V）

---

### 方法2：TXTファイルを取得（フルトランスクリプト）

**前提条件**
- ホストが「**フルトランスクリプト**」を有効にしている

**手順**

1. **会議中に文字起こしを表示**
   - 画面下部の「**字幕を表示**（CC）」→「**全文の文字起こしを表示**」

2. **文字起こしを保存**
   - 画面右下の「**文字起こしを保存**」をクリック

3. **保存先を確認**
   - **Windows**: `C:\Users\[ユーザー名]\Documents\Zoom`
   - **Mac**: `/Users/[ユーザー名]/Documents/Zoom`

4. **TXTファイルを開いてコピー**
   - テキストエディタで開く
   - 全選択→コピー
   - Teddyに貼り付け

## 得られる情報

| 情報 | 内容 |
|------|------|
| **会議情報** | 日時、参加者、議題を自動抽出 |
| **議事内容** | 話題ごとに整理された内容 |
| **決定事項** | ✅ マーク付きで明確に抽出 |
| **TODOリスト** | 担当者・期限付きで整理 |
| **次回予定** | 次回会議の情報 |

---

## その他の入力にも対応

| 入力タイプ | 説明 |
|-----------|------|
| **Zoom VTT** | **推奨** - クラウドレコーディングの字幕ファイル |
| **Zoom TXT** | **推奨** - フルトランスクリプトのテキストファイル |
| **Premiere Pro書き起こし** | 動画編集ソフトの書き起こし結果 |
| **音声認識アプリ** | Otter.ai、Nottaなどの出力 |
| **手動メモ** | 会議中にメモした内容 |
