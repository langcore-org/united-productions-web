# 出演者リサーチ

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

企画に合った**出演者候補を自動で探してくれる**機能です。プロフィール、出演実績、企画との相性まで分析して提案します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **企画に合うMCを探したい** | 「料理バラエティに合う、30代女性MCを探して」 |
| **専門家ゲストを探したい** | 「宇宙の専門家で、テレビ慣れしている人」 |
| **若手タレントを発掘したい** | 「バラエティ企画に合う若手俳優を探して」 |
| **特定の条件で絞りたい** | 「関西出身で、スポーツに詳しいタレント」 |

---

---

## 画面の使い方

1. **企画の内容や条件**を入力欄に書く
2. **送信ボタン**をクリック
3. AIが**候補者リスト**を作成
4. 気になる候補について**追加質問**できる

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **候補者リスト** | 条件に合う出演者を複数提案 |
| **プロフィール** | 年齢、職業、活動状況 |
| **出演実績** | 過去の番組出演歴 |
| **相性分析** | 企画とのマッチング評価 |
| **起用ポイント** | 交渉時の注意点など |

---

## 検索のコツ

| 指定すると良い条件 | 例 |
|------------------|-----|
| 年齢層 | 「30代」「20代後半〜30代前半」 |
| 性別 | 「女性」「男性」「男女ペア」 |
| 特技・専門 | 「料理が得意」「スポーツに詳しい」 |
| 雰囲気 | 「落ち着いた雰囲気」「明るく元気」 |
| 地域 | 「関西出身」「福岡出身」 |
