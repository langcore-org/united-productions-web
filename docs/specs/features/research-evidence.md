# エビデンスリサーチ

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

**情報の真偽を検証**し、信頼できる情報源を見つけ出す機能です。番組で使うデータや話題の情報をファクトチェックできます。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **統計データの出典を確認** | 「この視聴率データの出典を確認して」 |
| **SNSの噂を調べる** | 「SNSで話題のこの情報は本当？」 |
| **専門用語の解説を確認** | 「この医療用語の意味を正確に調べて」 |
| **歴史的事実を確認** | 「この出来事は実際に起きたこと？」 |
| **人物情報を確認** | 「この人物の経歴をファクトチェックして」 |

---

## 入力と出力の例

### 例：視聴率データの検証

**あなたの入力:**
```
「日本のテレビ視聴率は10年で半減した」という話を
聞いたのですが、本当でしょうか？
```

**AIの出力:**
```
【検証結果】

結論：部分的に正しい ⚠️

─────────────────────────

【詳細調査】

◆ 正確な情報
・世帯視聴率は確かに低下傾向にある
・ただし「半減」という表現は特定の指標を指している可能性が高い

◆ 注意が必要な点
・タイムシフト視聴、配信視聴を含めると、
  実際の「コンテンツ消費」は必ずしも減少していない
・定義（リアルタイム/タイムシフト/配信）によって数字が大きく変わる

─────────────────────────

【一次情報源】

✓ 日本民放連 放送世論調査
  https://www.j-ba.or.jp/
  → 信頼性：★★★★★

✓ 総務省 情報通信白書
  https://www.soumu.go.jp/
  → 信頼性：★★★★★

✓ NHK 放送文化研究所
  https://www.nhk.or.jp/bunken/
  → 信頼性：★★★★★

─────────────────────────

【番組制作上の注意】
・「視聴率」という言葉の定義を明確にしてから使用することを推奨
・最新の統合視聴率データを参照すること
```

---

## 画面の使い方

1. **検証したい情報**を入力欄に貼り付ける
2. **送信ボタン**をクリック
3. AIが**検証結果**と**情報源**を提示
4. 必要に応じて**追加質問**で深掘り

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **検証結果** | はい/いいえ/部分的に正しい |
| **詳細説明** | なぜその結論になったか |
| **一次情報源** | 信頼できる出典リスト（URL付き） |
| **信頼性評価** | 情報源ごとの信頼度 |
| **注意点** | 番組で使用する際の注意事項 |

---

## 検証のポイント

| 検証レベル | 説明 |
|-----------|------|
| **はい** | 複数の一次情報源で確認済み |
| **いいえ** | 誤情報であることが確認できた |
| **部分的に正しい** | 文脈や条件によって解釈が異なる |
