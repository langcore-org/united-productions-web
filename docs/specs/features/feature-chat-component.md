# FeatureChatコンポーネント仕様

> **最終更新**: 2026-02-25 13:15

---

## 概要

各機能ページで共通して使用するチャットUIコンポーネント。

---

## コンポーネント詳細

| 項目 | 内容 |
|------|------|
| **コンポーネント名** | `FeatureChat` |
| **ファイルパス** | `components/ui/FeatureChat.tsx` |
| **用途** | 機能別チャットUIの共通実装 |

---

## Propsインターフェース

```typescript
interface FeatureChatProps {
  featureId: string;                    // 機能識別子
  chatId?: string;                      // チャットセッションID（指定なしで新規チャット）
  onChatCreated?: (chatId: string) => void;  // 新規チャット作成時のコールバック
  title: string;                        // ページタイトル
  systemPrompt: string;                 // システムプロンプト
  placeholder: string;                  // 入力欄プレースホルダー
  inputLabel?: string;                  // 入力エリアラベル
  outputFormat?: "markdown" | "plaintext";  // 出力形式（デフォルト: markdown）
}
```

## 入出力仕様

### ユーザー入力

| 入力タイプ | 内容 |
|-----------|------|
| **テキスト入力** | チャットメッセージ（テキストエリア） |
| **送信ボタン** | メッセージ送信 |
| **新規チャットボタン** | 新しいチャットセッション開始 |
| **コピーボタン** | plaintextモード時の全文コピー |

### AI出力

| 出力タイプ | 内容 |
|-----------|------|
| **ストリーミング表示** | SSEによるリアルタイム文字表示 |
| **Markdown表示** | 書式付きテキスト（表、リスト、コードブロック等） |
| **Plaintext表示** | 書式なしテキスト（コピーボタン付き） |
| **エラー表示** | エラーメッセージ |

## 機能一覧

| 機能 | 説明 |
|------|------|
| **ストリーミングレスポンス** | SSEによるリアルタイム表示 |
| **会話履歴保存** | Prismaによる自動保存 |
| **チャットセッション管理** | chatIdベースのCRUD |
| **自動タイトル生成** | 新規チャット時にGrokでタイトル生成 |
| **コピー機能** | plaintextモード時のワンクリックコピー |
| **システムプロンプト切り替え** | 機能別のプロンプト適用 |

---

## 出力形式

### markdown（デフォルト）

- Markdownレンダリング対応
- コードブロック、リスト、表などを表示
- シンタックスハイライト

### plaintext

- 書式なしテキスト表示
- **コピーボタン**表示
- Word貼り付け対応

## 自動タイトル生成

新規チャット時（chatId未指定時）の動作：

1. ユーザーが最初のメッセージを送信
2. AIが応答を生成
3. バックグラウンドでタイトル生成APIを呼び出し
4. 生成されたタイトルをチャットセッションに保存

```typescript
// タイトル生成プロンプト
const titleGenerationPrompt = `
以下の会話内容から、30文字以内で適切なタイトルを生成してください。
タイトルは日本語で、内容を簡潔に表現してください。

会話内容:
{messages}

タイトル:
`;
```

  title="出演者リサーチ"
