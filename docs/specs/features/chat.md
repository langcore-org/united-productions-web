# チャット機能

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

何でも気軽にAIに相談できる**汎用チャット**です。特定のフォーマットに縛られず、自由に質問や相談ができます。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **業界情報を調べたい** | 「今週の放送業界のトピックを教えて」 |
| **文章を校正したい** | 「この企画書の文章をもっとわかりやすくしてください」 |
| **アイデアを出したい** | 「企画のオープニングでインパクトのある演出を考えて」 |
| **情報を整理したい** | 「この打ち合わせメモを整理して」 |
| **最新ニュースを知りたい** | 「最近話題のSNSトレンドを教えて」 |

---

## 入力と出力の例

### 例1：文章の校正

**あなたの入力:**
```
この企画は視聴者の興味を引く内容になっており、
面白いと思います。ぜひ採用してください。
```

**AIの出力:**
```
【修正案】
本企画は視聴者の関心を惹く内容となっており、
高いエンターテイメント性を備えています。
ご採用をご検討いただけますと幸いです。

【修正ポイント】
・「今回」→「本」：よりフォーマルな表現に
・「面白い」→「高いエンターテイメント性」：具体的に
・「ぜひ採用してください」→「ご検討いただけますと」：謙譲表現
```

### 例2：アイデア出し

**あなたの入力:**
```
料理番組のオープニングで、視聴者の注目を集める
演出のアイデアを3つ出してください
```

**AIの出力:**
```
【オープニング演出アイデア】

1. 「食材の旅」演出
　・カメラが食材の産地を追いかける
　・農家さんのこだわりを紹介しつつ料理へ繋ぐ

2. 「タイムラプス料理」
　・完成料理から逆再生で過程を示す
　・「どうやって作った？」と興味を引く

3. 「MCの失敗談」
　・料理失敗エピソードからスタート
　・「今日は成功させるぞ！」という展開
```

---

## 画面の使い方

1. **入力欄**に質問や相談を入力
2. **送信ボタン**をクリック
3. AIが**リアルタイム**で回答を表示
4. 必要に応じて**追加質問**を続ける

---

## 特徴

| 特徴 | 説明 |
|------|------|
| **自由形式** | 決まった形式なく、なんでも相談できる |
| **最新情報** | 必要に応じてWeb検索・X検索を実行 |
| **継続対話** | 前後の文脈を理解した上で回答 |
