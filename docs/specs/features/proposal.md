# 新企画立案

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

**新しい番組企画をAIが提案**してくれます。番組設定に登録した情報を参考に、企画書形式でアイデアをまとめてくれます。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **新企画のアイデアがない** | 「若年層向けの新しい企画を考えて」 |
| **既存企画のバリエーション** | 「過去の企画をアレンジした新案を出して」 |
| **特番企画を考えたい** | 「春の特番企画を提案して」 |
| **具体的な条件で提案** | 「予算100万円以内で収録できる企画」 |

---

## 画面の使い方

1. **企画の方向性**を入力欄に書く
2. **送信ボタン**をクリック
3. AIが**企画書形式**で提案
4. 気に入らない点は**追加指示**で修正

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **タイトル案** | 複数の候補を提案 |
| **コンセプト** | 目的・ターゲット・狙い |
| **内容構成** | 番組の流れ・コーナー構成 |
| **出演者提案** | MC・レギュラー・ゲスト候補 |
| **制作ポイント** | 予算・セット・注意点 |
| **差別化** | 類似番組との違い |

---

## より良い提案を得るコツ

| 指定すると良い条件 | 例 |
|------------------|-----|
| ターゲット層 | 「20代女性」「ファミリー層」 |
| ジャンル | 「料理」「旅」「クイズ」 |
| 予算規模 | 「低予算」「特番規模」 |
| 時間帯 | 「ゴールデン」「深夜」 |
| 目的 | 「視聴率」「SNS拡散」「商品PR」 |
