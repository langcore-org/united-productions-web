# NA原稿作成

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

**文字起こしからナレーション（NA）原稿を自動作成**します。話し言葉を原稿用に整え、尺に合わせて調整。Wordにそのまま貼り付けられる形式で出力します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **Premiere Proの書き起こしを原稿に** | Premiere Proの書き起こし結果を貼り付ける |
| **フィラーを除去したい** | 「えーと」「あの」などを除去 |
| **話者を判定して整形** | 複数人の発言を整理 |
| **尺に合わせて調整** | 「30秒程度にまとめて」 |

---

## 入力と出力の例

### 例：番組オープニングのNA原稿

**あなたの入力:**
```
以下の文字起こしをNA原稿にしてください。
尺は30秒程度でお願いします。

---

えーと、今日はですね、新しい料理番組の紹介を、
あの、したいと思います。で、えー、この番組では、
まあ、色々な料理人が、えー、登場しまして、
それぞれの、あの、得意料理を、えー、紹介していきます。
はい、えー、それでは、どうぞ。
```

**AIの出力:**
```
【NA原稿】

新しい料理番組のご紹介。
この番組では、さまざまな料理人が登場し、
それぞれの得意料理を紹介していきます。
さあ、それではご覧ください。

---
【情報】
・想定尺：28秒
・文字数：72文字
・トーン：明るく、親しみやすい案内役
```

---

## 画面の使い方

1. **文字起こしテキスト**を入力欄に貼り付ける
2. **尺や調整内容**を指示（オプション）
3. **送信ボタン**をクリック
4. **コピーボタン**で原稿をコピー
5. Wordなどに**貼り付け**

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **NA原稿** | 整えられたナレーション原稿 |
| **想定尺** | 読み上げ時間の目安 |
| **文字数** | 原稿の文字数 |
| **トーン** | 読み方のイメージ |

---

## コピー機能

**「コピー」ボタン**を押すと、原稿部分が自動でコピーされます。

- 書式なしテキストとしてコピーされる
- Wordに貼り付けても書式が崩れない
- 改行が保持される

---

## 入力のコツ

| 指示例 | 効果 |
|--------|------|
| 「30秒程度にまとめて」 | 尺に合わせて調整 |
| 「フィラーを除去して」 | 「えーと」「あの」などを削除 |
| 「話者を判定して」 | 複数人の発言を整理 |
| 「敬語に直して」 | 丁寧な表現に変更 |
| 「もっと明るいトーンで」 | 雰囲気を調整 |
