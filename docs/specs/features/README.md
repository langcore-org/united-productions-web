## Teddyでできること

Teddyはテレビ制作現場の業務をAIで支援するツールです。以下のことができます：

---

## 機能一覧

### 💬 チャット機能

| 機能 | できること | 使いシーン |
|------|-----------|-----------|
| [汎用チャット](./chat.md) | 何でも気軽にAIに相談 | 文章の校正、アイデア出し、業界情報収集 |
| [出演者リサーチ](./research-cast.md) | 企画に合う出演者を提案してもらう | MC探し、専門家ゲスト探し、若手発掘 |
| [エビデンスリサーチ](./research-evidence.md) | 情報の真偽を検証 | 統計データ確認、SNS噂の事実確認 |
| [議事録作成](./minutes.md) | 文字起こしから議事録を自動作成 | Zoom会議の記録、打ち合わせ内容の整理 |
| [新企画立案](./proposal.md) | 新しい番組企画を提案してもらう | アイデアがない時、既存企画のバリエーション |
| [NA原稿作成](./na-script.md) | 文字起こしからナレーション原稿を作成 | Premiere Pro書き起こしの整形、フィラー除去 |

### 📋 管理機能

| 機能 | できること | 使いシーン |
|------|-----------|-----------|
| [チャット履歴](./chat-history.md) | 過去のAI相談を見返す | 以前の相談内容の確認、途中から再開 |

---

## 入出力まとめ

| 機能 | あなたが入力するもの | AIが出力するもの |
|------|-------------------|-----------------|
| **汎用チャット** | 質問・相談・依頼 | 回答・アドバイス・アイデア |
| **出演者リサーチ** | 企画内容・探したい条件 | 出演者候補リスト（プロフィール・実績付き） |
| **エビデンスリサーチ** | 検証したい情報・主張 | 検証結果・一次情報源・信頼性評価 |
| **議事録作成** | 文字起こしテキスト | 整形された議事録（決定事項・TODO付き） |
| **新企画立案** | 企画の方向性・テーマ | 企画書形式の提案（タイトル・構成・出演者） |
| **NA原稿作成** | 文字起こしテキスト | ナレーション原稿（Wordコピー対応） |

## 画面の構成

### サイドバー（左側）

```
Teddy

【新規作成】
💬 チャット
👥 出演者リサーチ
🛡️ エビデンスリサーチ
📝 議事録作成
💡 新企画立案
🎤 NA原稿作成

【履歴】
🕐 履歴を見る

【その他】
🚪 ログアウト
```

### メインエリア（中央）

- **入力欄**: 下側に配置されたテキストエリア
- **送信ボタン**: 入力欄の横
- **会話表示**: 中央にバブル形式で表示
- **コピーボタン**: NA原稿作成などで表示

---

## 使い方の流れ

### 初めて使う時

1. サイドバーからやりたい機能を選択
2. 入力欄に内容を入力
3. 送信ボタンをクリック
4. AIの回答を待つ（リアルタイムで表示）

### 続きから再開する時

1. サイドバーの「履歴を見る」をクリック
2. 過去の会話リストから選ぶ
3. その時点の会話を再開
