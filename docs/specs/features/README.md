# 機能ガイド

> **最終更新**: 2026-02-25 13:50

---

## Teddyでできること

Teddyはテレビ制作現場の業務をAIで支援するツールです。

---

## 機能ロードマップ

### ✅ 現時点で利用可能

現在すぐに使える機能です。

| 機能 | できること | 使いシーン |
|------|-----------|-----------|
| [汎用チャット](./chat.md) | 何でも気軽にAIに相談 | 文章の校正、アイデア出し、業界情報収集 |
| [議事録作成](./minutes.md) | 文字起こしから議事録を自動作成 | Zoom会議の記録、打ち合わせ内容の整理 |
| [新企画立案](./proposal.md) | 新しい番組企画を提案してもらう | アイデアがない時、既存企画のバリエーション |
| [NA原稿作成](./na-script.md) | 文字起こしからナレーション原稿を作成 | Premiere Pro書き起こしの整形、フィラー除去 |
| [出演者リサーチ](./research-cast.md) | 企画に合う出演者を提案してもらう | MC探し、専門家ゲスト探し、若手発掘 |
| [エビデンスリサーチ](./research-evidence.md) | 情報の真偽を検証 | 統計データ確認、SNS噂の事実確認 |
| [チャット履歴](./chat-history.md) | 過去のAI相談を見返す | 以前の相談内容の確認、途中から再開 |

---

### ⏳ いつか対応（時期未定）

将来的に検討している機能です。優先度・実現可能性を検討中。

| 機能 | 検討内容 | 検討理由 |
|------|---------|---------|
| **情報リサーチ** | トピックやテーマの情報を収集 | 企画ネタ探しに有用だが、現時点では汎用チャットで代替可能 |
| **構成資料作成** | 企画書・構成表の自動生成 | Word/PowerPointへのAI出力はレイアウト崩れが生じやすく、技術的課題あり |
| **Word直接エクスポート** | NA原稿などをWord形式で直接保存 | 現在はコピー＆ペースト方式。将来的な改善候補 |
| **音声入力対応** | 音声から直接文字起こし | データ容量・処理負荷・コストの課題あり |
| **番組設定** | 番組情報・過去企画の管理 | 新企画立案の精度向上のため検討していたが、現時点では不要と判断 |

## 各機能の入力と出力

### 💬 汎用チャット

| 項目 | 内容 |
|------|------|
| **入力** | 自由テキスト（質問・相談・依頼） |
| **入力例** | 「今週の放送業界のトピックを教えて」「この文章を校正してください」 |
| **出力** | Markdown形式の回答・アドバイス・アイデア |
| **出力例** | 業界ニュースまとめ、校正結果（修正案と修正ポイント） |

---

### 📝 議事録作成

| 項目 | 内容 |
|------|------|
| **入力** | 文字起こしテキスト（Zoom、Premiere Pro、手動メモなど） |
| **入力例** | 「会議日：2月20日 参加者：山田、佐藤...」 |
| **出力** | Markdown形式の構造化された議事録 |
| **出力例** | 会議情報、議事内容、決定事項✅、TODOリスト、次回予定 |

---

### 💡 新企画立案

| 項目 | 内容 |
|------|------|
| **入力** | 企画の方向性・テーマ・条件 |
| **入力例** | 「20代〜30代向けに、料理×バラエティの新しい企画を考えて」 |
| **出力** | Markdown形式の企画書 |
| **出力例** | タイトル案、コンセプト、内容構成、出演者提案、制作ポイント、差別化 |

---

### 🎤 NA原稿作成

| 項目 | 内容 |
|------|------|
| **入力** | 文字起こしテキスト＋尺などの指示（オプション） |
| **入力例** | 「以下の文字起こしをNA原稿にしてください。尺は30秒程度で」 |
| **出力** | Plaintext形式のナレーション原稿 |
| **出力例** | 【NA原稿】本文、【情報】想定尺・文字数・トーン |

---

### 👥 出演者リサーチ

| 項目 | 内容 |
|------|------|
| **入力** | 企画内容・探したい出演者の条件 |
| **入力例** | 「料理バラエティに出てほしい、30代女性で料理が得意な人」 |
| **出力** | Markdown形式の出演者候補リスト |
| **出力例** | 表形式の候補者リスト（名前・年齢・特技・相性評価）、詳細プロフィール、起用ポイント |

---

### 🛡️ エビデンスリサーチ

| 項目 | 内容 |
|------|------|
| **入力** | 検証したい情報・主張 |
| **入力例** | 「日本のテレビ視聴率は10年で半減したという話は本当？」 |
| **出力** | Markdown形式の検証結果 |
| **出力例** | 結論（はい/いいえ/部分的に正しい）、詳細説明、一次情報源リスト、信頼性評価 |

---

### 🕐 チャット履歴

| 項目 | 内容 |
|------|------|
| **入力** | UI操作（一覧表示、検索、削除、再開） |
| **出力** | リストUI（タイトル、アイコン、更新日時、プレビュー） |

---

### 🔍 情報リサーチ（⏳ いつか対応）

| 項目 | 内容 |
|------|------|
| **入力** | 調べたいテーマ・トピック |
| **入力例** | 「アジフライに関する雑学やトリビアを教えて」 |
| **出力** | Markdown形式の情報まとめ |
| **出力例** | 基本情報、トリビア、最新ニュース、企画活用アイデア |

## 使い方の流れ

### 初めて使う時

1. サイドバーからやりたい機能を選択
2. 入力欄に内容を入力
3. 送信ボタンをクリック
4. AIの回答を待つ（リアルタイムで表示）

### 続きから再開する時

1. サイドバーの「履歴を見る」をクリック
2. 過去の会話リストから選ぶ
3. その時点の会話を再開
