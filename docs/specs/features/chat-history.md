# チャット履歴

> **最終更新**: 2026-02-25 13:20

---

## この機能でできること

**過去にAIと話した内容を見返したり、再開したり**できます。いつでも過去の相談内容を確認できます。

---

## こんな時に使う

| シーン | 使い方 |
|--------|--------|
| **以前の相談を見返したい** | 履歴リストから選択 |
| **途中までの相談を再開** | 該当チャットをクリック |
| **不要な履歴を削除** | 削除ボタンをクリック |
| **探している相談を検索** | 検索機能でキーワード検索 |

## できる操作

| 操作 | 方法 | 結果 |
|------|------|------|
| **履歴を開く** | リストをクリック | その時点の会話を再開 |
| **履歴を削除** | 🗑️ アイコンをクリック | 削除確認後、削除 |
| **検索** | 検索欄にキーワード入力 | 該当する履歴を絞り込み |

---

## 表示される情報

| 情報 | 説明 |
|------|------|
| **アイコン** | どの機能で相談したか（👥📝💡など） |
| **タイトル** | AIが自動生成した会話のタイトル |
| **日時** | 最後に更新した日時 |
| **プレビュー** | 会話内容の一部（最初のメッセージ） |

---

## 自動タイトルについて

新規チャット時に、AIが自動でタイトルを付けます：

- 会話の内容を分析して適切なタイトルを生成
- 30文字以内の分かりやすい表現
- 後からタイトルを変更することも可能
