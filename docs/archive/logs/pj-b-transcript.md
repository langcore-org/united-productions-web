# PJ-B 起こし・NA原稿機能 実装ログ

## 概要
Premiere Pro書き起こしテキストをAIで整形し、NA原稿を作成する機能。

## 実装内容

### 2024-02-16

#### 1. プロンプトファイル作成
- **ファイル**: `prompts/transcript-format.ts`
- **内容**: 
  - 話者ラベル定義（櫻井/末澤/N/その他）
  - システムプロンプト（整形ルール）
  - ユーザープロンプト生成関数

#### 2. APIルート作成
- **ファイル**: `app/api/transcripts/route.ts`
- **内容**:
  - POSTエンドポイント
  - Zodバリデーション
  - Gemini 2.5 Flash-Lite統合
  - エラーハンドリング

#### 3. レイアウト作成
- **ファイル**: `app/transcripts/layout.tsx`
- **内容**:
  - メタデータ設定
  - Sidebar統合
  - ダークテーマ適用

#### 4. メインページ作成
- **ファイル**: `app/transcripts/page.tsx`
- **内容**:
  - 話者ラベル表示（櫻井/末澤/N/その他）
  - 入力エリア（高さ400px）
  - AI整形ボタン
  - Markdownプレビュー
  - コピー機能

## 技術仕様

### LLM
- **モデル**: Gemini 2.5 Flash-Lite
- **フォールバック**: Gemini 3.0 Flash

### UI仕様
- **背景**: #0d0d12
- **カード**: #1a1a24
- **アクセント**: #ff6b00
- **入力エリア高さ**: 400px

### 話者ラベル
| ラベル | 説明 | カラー |
|--------|------|--------|
| 櫻井 | 櫻井翔 | 青 |
| 末澤 | 末澤誠也 | 緑 |
| N | ナレーション | 紫 |
| その他 | その他出演者 | 灰 |
