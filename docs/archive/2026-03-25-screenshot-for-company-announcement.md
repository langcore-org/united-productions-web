## 発表内容

島田源太郎が社内朝礼で以下を発表：
- 3月末にローンチ予定であること
- リリース機能：リサーチ、議事録作成、新規企画立案
- **ナレーション原稿作成は省いたラインナップ**

| **機能紹介** | サイドバー表示（全機能アイコン） | PNG/JPG | ✅ 提供済み |

## スクリーンショット仕様

### メイン画面（チャット）

```
【表示内容】
- チャットインターフェース
- サイドバー（全機能が見える状態）
- 空のチャットエリア
- 入力欄

【注意点】
- ナレーション原稿作成アイコンは非表示
- 清潔感のある状態
- 機密情報が映っていないこと
```

### サイドバー表示

```
【表示する機能アイコン】
1. チャット
2. 出演者リサーチ
3. エビデンスリサーチ
4. 議事録作成
5. 新企画立案
6. 履歴

【非表示】
- ナレーション原稿作成
```
