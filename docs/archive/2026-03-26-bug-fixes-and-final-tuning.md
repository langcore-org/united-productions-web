## ロールバック計画

### 緊急時対応

万が一、リリース後に重大な問題が発見された場合：

```
1. 問題の切り分け（本番環境の問題か確認）
2. ホットフィックスの検討
3. ホットフィックスが難しい場合はロールバック
4. ステージング環境での検証
5. 再デプロイ
```
