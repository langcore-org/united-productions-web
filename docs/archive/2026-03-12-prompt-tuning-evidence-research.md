## エビデンスリサーチの定義

この機能で実現したいこと：
- 企画・企画書作成に必要な検証データの収集
- 信頼性の高い情報源の提示
- 出典付きの事実関係の整理

## プロンプト設計

### 1. 入力情報

```
【入力】
- リサーチテーマ/キーワード
- 番組情報（選択された場合）
- 必要なエビデンスの種類（統計、事例、専門家意見等）
```

### 2. 出力形式

```
【出力形式】
1. エビデンス概要
   - テーマに関する主要な事実関係

2. 詳細エビデンス
   a) 統計データ（あれば）
      - 数値
      - 出典
      - 調査年月日
   
   b) 具体的事例
      - 事例の概要
      - 関連する番組/メディア
      - 出典
   
   c) 専門家コメント/論文（あれば）
      - 専門家名/論文タイトル
      - 要約
      - 出典

3. 信頼性評価
   - 情報源の信頼性レベル
   - 注意点（あれば）

4. 出典一覧
   - [1] [2] ... の形式
```

### 3. 検索指示

```
検索戦略:
1. 公式統計（政府機関、業界団体）を優先
2. 信頼性の高いメディア報道
3. 学術論文・専門家意見
4. SNS上の話題性（補助的に）

注意事項:
- 出典不明の情報は明記する
- 古い情報は年月日を明示
- 複数の情報源で cross-check
```
