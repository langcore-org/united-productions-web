# ロケ地リサーチ

> **最終更新**: 2026-02-25 13:35

---

## この機能でできること

**撮影に適したロケーション・ロケ地を提案**してくれます。撮影許可、アクセス、周辺情報なども含めて調査します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **ロケ地を探したい** | 「東京で昭和レトロな雰囲気の店を探して」 |
| **撮影許可の情報が欲しい** | 「この公園で撮影するには許可が必要？」 |
| **宿泊施設を探したい** | 「ロケクルーが泊まれる近くのホテル」 |
| **類似ロケを探したい** | 「この雰囲気に似た別の場所を探して」 |

---

## 画面の使い方

1. **ロケ地の条件**を入力欄に書く（場所、雰囲気、用途など）
2. **送信ボタン**をクリック
3. AIが**候補地リスト**を作成
4. 気になる場所について**詳細情報**を追加で質問

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **候補地リスト** | 条件に合うロケ地を複数提案 |
| **基本情報** | 住所、営業時間、連絡先 |
| **撮影情報** | 許可の有無、料金、おすすめ時間帯 |
| **アクセス** | 最寄駅、駐車場、機材搬入経路 |
| **周辺情報** | 宿泊施設、飲食店、コンビニなど |
| **注意点** | 撮影上のポイント、リスク |

## 検索のコツ

| 指定すると良い条件 | 例 |
|------------------|-----|
| **エリア** | 「東京23区内」「関東圏」「地方都市」 |
| **雰囲気** | 「昭和レトロ」「モダン」「自然」 |
| **用途** | 「料理撮影」「インタビュー」「ロケ」 |
| **規模** | 「大人数」「小規模」「貸切可能」 |
| **予算** | 「無料」「低予算」「料金不問」 |
