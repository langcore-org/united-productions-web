# 情報リサーチ

> **最終更新**: 2026-02-25 13:35

---

## この機能でできること

**トピックやテーマに関する情報を収集・整理**してくれます。雑学、トリビア、最新ニュース、統計データなどをまとめて提供します。

---

## こんな時に使う

| シーン | 入力例 |
|--------|--------|
| **企画のネタを探したい** | 「アジフライに関する雑学やトリビアを教えて」 |
| **最新情報を調べたい** | 「最近話題の食品トレンドを調べて」 |
| **専門用語を理解したい** | 「発酵食品の種類と特徴を解説して」 |
| **統計データを探したい** | 「日本の料理番組視聴率の推移を調べて」 |

---

## 入力と出力の例

### 例：アジフライの情報収集

**あなたの入力:**
```
アジフライに関する雑学、トリビア、歴史などを教えて。
企画で使える面白い情報があれば教えてほしい。
```

**AIの出力:**
```
【アジフライ 情報まとめ】

━━━━━━━━━━━━━━━━━━━━━
基本情報
━━━━━━━━━━━━━━━━━━━━━
・アジフライ（鯵フライ）は、日本の洋食メニューの定番
・主に真あじを使用し、パン粉をまぶして揚げる
・1960年代頃から一般的になった家庭料理

━━━━━━━━━━━━━━━━━━━━━
面白いトリビア
━━━━━━━━━━━━━━━━━━━━━

1. 【地域差】
   ・関西では「アジフライ」、関東では「あじフライ」と表記
   ・九州では「あいごフライ」（方言）とも呼ばれる

2. 【食文化】
   ・昭和の大衆食堂文化の象徴的メニュー
   ・おかずとしてだけでなく、おつまみとしても人気

3. 【栄養】
   ・DHA・EPAが豊富
   ・カルシウムも摂取できる（骨まで食べる場合）

━━━━━━━━━━━━━━━━━━━━━
企画活用アイデア
━━━━━━━━━━━━━━━━━━━━━

□ 「全国アジフライ対決」
  → 各地域の独自アレンジを比較

□ 「アジフライの進化系」
  → 高級店のアジフライ vs 大衆食堂のアジフライ

□ 「アジフライ職人」
  → 揚げ方のプロを探す
```

---

## 画面の使い方

1. **調べたいテーマやトピック**を入力欄に書く
2. **送信ボタン**をクリック
3. AIが**情報を収集・整理**して提示
4. 気になる点について**追加質問**できる

---

## 得られる情報

| 情報 | 内容 |
|------|------|
| **基本情報** | テーマの概要、歴史、背景 |
| **トリビア・雑学** | 企画に使える面白い情報 |
| **最新ニュース** | トピックに関する最近の動向 |
| **統計データ** | 数字に基づく客観的情報 |
| **企画アイデア** | 情報を活かした企画案の提案 |

---

## 検索のコツ

| 指定すると良い条件 | 例 |
|------------------|-----|
| **深さ** | 「概要だけ」「詳しく」「専門的に」 |
| **観点** | 「歴史」「文化」「経済」「科学」 |
| **用途** | 「企画で使う」「ナレーション用」「クイズに使う」 |
| **地域** | 「日本」「海外」「地域比較」 |

## 情報源

| 情報源 | 説明 |
|--------|------|
| **Web検索** | 一般的なウェブ情報 |
| **SNS検索** | X/Twitter、Instagram、YouTube、TikTokの最新情報 |
| **ニュース** | 最新の記事・プレスリリース |
| **学術情報** | 論文・研究結果（必要に応じて） |
