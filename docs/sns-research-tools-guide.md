# SNSリサーチツール・API・サービス包括調査ガイド

## 概要

本ドキュメントは、SNS（ソーシャルメディア）リサーチ機能の強化に向けた各種ツール、API、サービスの包括的な調査結果です。

---

## 1. 公式API

### 1.1 Twitter/X API

| 項目 | 詳細 |
|------|------|
| **価格体系** | **Pay-per-usage方式**（クレジット制）<br>• Free Tier: 月額無料（制限あり）<br>• Basic: $100/月<br>• Pro: $5,000/月<br>• Enterprise: カスタム料金 |
| **レート制限** | • Free: 24時間ウィンドウでエンドポイントごとに制限<br>• Basic/Pro: 15分ウィンドウでの制限<br>• Enterprise: カスタム制限 |
| **取得可能データ** | • ツイート検索・検索<br>• ユーザー情報・フォロー管理<br>• エンゲージメント指標（いいね、リツイート、返信）<br>• トレンド分析<br>• DM・スペース管理<br>• 会話トラッキング・編集履歴 |
| **利用規約の制約** | • 公開データのみアクセス可能<br>• プライベート指標は自分のアカウントのみ<br>• 同一リソースは24時間以内の再リクエストは無料<br>• 月間200万ポスト読み取り上限あり |
| **申請プロセス** | 1. console.x.com でサインアップ<br>2. アプリ作成と認証情報取得<br>3. クレジット購入<br>4. 即時利用開始 |
| **長所** | • リアルタイムデータ取得可能<br>• 柔軟なフィールド選択<br>• 注釈機能（人物・場所・製品・トピックの自動識別） |
| **短所** | • 2023年以降、無料枠が大幅縮小<br>• 高額な料金（月$100〜$5,000）<br>• 過去データ（Historical）アクセスにはEnterpriseプランが必要 |

**推奨用途**: リアルタイムモニタリング、トレンド分析、大規模データ収集（予算がある場合）

### 1.2 Instagram Graph API

| 項目 | 詳細 |
|------|------|
| **価格体系** | **無料**（Facebook開発者アカウントが必要） |
| **レート制限** | • Business Use Case (BUC) レート制限適用<br>• DMは1時間あたり200件まで |
| **取得可能データ** | • プロフィール情報<br>• メディア（投稿、リール、ストーリー）<br>• コメント、いいね、保存数<br>• インサイト（インプレッション、リーチ、フォロワー推移）<br>• DM管理（ビジネスアカウントのみ） |
| **利用規約の制約** | • ビジネスアカウントまたはクリエイターアカウントが必要<br>• 個人アカウントは非対応<br>• ストーリーは24時間以内に取得必須 |
| **申請プロセス** | 1. Facebook Developersでアプリ作成<br>2. ビジネス認証<br>3. アプリ審査（公開する場合） |
| **長所** | • 無料で利用可能<br>• 詳細なインサイトデータ取得可能 |
| **短所** | • 個人アカウント非対応<br>• Basic Display API廃止で選択肢が狭まった<br>• ハッシュタグ検索に制限あり |

### 1.4 TikTok Research API

| 項目 | 詳細 |
|------|------|
| **価格体系** | **無料**（研究者向け） |
| **レート制限** | 審査後に通知 |
| **取得可能データ** | • アカウント情報（プロフィール、フォロワー/フォロー一覧、いいね動画）<br>• コンテンツ（動画情報、メタデータ）<br>• ショップ情報 |

| **長所** | • 無料で大規模データにアクセス可能<br>• 研究目的に適した構造化データ |
| **短所** | • 研究者のみがアクセス可能（商用不可）<br>• 審査に時間がかかる |

**推奨用途**: 学術研究、論文執筆、トレンド分析

## 重要な注意事項

### 法的コンプライアンス

1. **利用規約の遵守**: すべてのSNSプラットフォームは利用規約でスクレイピングを禁止している場合があります。公式APIまたは合法的なデータプロバイダーの使用を推奨します。

2. **プライバシー規制**: GDPR、個人情報保護法などの規制に注意が必要です。

3. **データの正確性**: サードパーティデータは推定値を含む場合があります。

4. **APIの変更**: SNSプラットフォームは頻繁にAPIを変更・制限します。

### 推奨アプローチ

```
フェーズ1: 無料ツールで検証
  ↓ YouTube API + Social Blade + HINOME
フェーズ2: 有料APIで拡張
  ↓ Twitter API Basic + BuzzSumo
フェーズ3: エンタープライズツールで本格化
  ↓ Brandwatch or Talkwalker
```
