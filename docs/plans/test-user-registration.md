## 背景

現在のアプリはテスト段階にあり、Googleの管理画面でテストユーザーとして登録しないとログインできない状況。

「もしすぐ触りたかったらテストユーザー登録をお渡しします」とのことで、希望者がいれば対応する必要がある。

## テストユーザー登録手順

### 1. Google Cloud Consoleでの操作

```
1. Google Cloud Consoleにログイン
2. 「APIとサービス」→「OAuth同意画面」を選択
3. 「テストユーザー」セクションを展開
4. 「ユーザーを追加」をクリック
5. 追加したいユーザーのメールアドレスを入力
6. 保存
```

### 2. 必要な情報

テストユーザー登録に必要な情報：
- メールアドレス（Googleアカウント）
- 名前（管理用）

## 対応フロー

### テストユーザー登録希望時

```
1. United Productions様から登録希望者の連絡を受ける
   - 名前
   - メールアドレス

2. Google Cloud Consoleでテストユーザーに追加

3. 登録完了を連絡

4. ユーザーに以下を案内
   - アプリURL
   - ログイン方法（Google認証）
```
