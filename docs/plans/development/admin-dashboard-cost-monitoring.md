# 管理画面調整（コスト監視）

> **優先度**: 🟢 低  
> **期限**: 2026-03-19  
> **担当**: 浅野宏耀  
> **ステータス**: 進行中

## 目的

AI利用コストの監視機能を整備し、異常なコスト発生を早期に検知できるようにする。

## 現状

- 管理画面は実装済み
- トークン使用量の表示機能あり
- 正確性の調整が必要

## タスク内容

### 1. コスト計算の修正

**現在の課題**:
- 1発で10ドルと表示されるなど、明らかにおかしい計算結果

**修正内容**:
- [ ] トークン数→コスト計算ロジックの見直し
- [ ] Grok APIの課金体系に基づく正確な計算
- [ ] 通貨換算（USD→JPY）の確認

### 2. 監視機能の強化

- [ ] ユーザー別使用量の集計
- [ ] 日次/週次/月次の使用量レポート
- [ ] 異常検知（急激な使用量増加のアラート）

### 3. UI改善

- [ ] コスト表示の単位整備
- [ ] グラフ表示（使用量推移）
- [ ] エクスポート機能
