# SNS有料ツール統合検証

> **優先度**: 🔴 高  
> **期限**: 2026-03-05  
> **担当**: 浅野宏耀  
> **ステータス**: 🟡 **進行中（調査開始）**  
> **親タスク**: [実装プラン 2026年3月](../implementation-plan-2026-03.md)

---

## 目的

YouTubeとInstagramのリサーチ統合に必要な有料ツールの検証・選定を行い、技術的・費用的な実現可能性を確認する。

## 背景

- X（旧Twitter）のリサーチ統合は既存のGrokモデルの機能として実装済み
- YouTubeとInstagramについては、Xとは別途有料ツールの統合が必要
- TikTokはAPI申請に時間がかかるため、3月中の実装は見送り

## 検討対象ツール

### 候補1: YouTube Data API（直接統合）

| 項目 | 内容 |
|------|------|
| **提供元** | Google |
| **料金** | クォータ制（無料枠あり、超過分は有料） |
| **技術難易度** | 中 |
| **データ精度** | 高（公式API） |
| **メリット** | 信頼性高、コスト制御可能 |
| **デメリット** | 申請・設定に手間がかかる |

### 候補2: Instagram Graph API（直接統合）

| 項目 | 内容 |
|------|------|
| **提供元** | Meta |
| **料金** | 基本無料（レート制限あり） |
| **技術難易度** | 中〜高 |
| **データ精度** | 中（ビジネスアカウントのみ対象の制限あり） |
| **メリット** | 公式API |
| **デメリット** | 取得できる情報に制限あり |

### 候補3: 第三者SNS分析ツール

#### 3-1. Brandwatch

| 項目 | 内容 |

| **対応SNS** | YouTube, Instagram, X等 |
| **技術難易度** | 中（API提供あり） |
| **メリット** | 包括的な分析機能 |
| **デメリット** | コストが高い |

#### 3-2. Sprout Social

| 項目 | 内容 |
|------|------|
| **料金** | 中額（月額数万円〜） |
| **対応SNS** | YouTube, Instagram, X等 |
| **技術難易度** | 低（API提供あり） |
| **メリット** | 使いやすいAPI |
| **デメリット** | 分析機能はシンプル |

#### 3-3. Hootsuite

| 項目 | 内容 |
|------|------|
| **料金** | 中額（月額数万円〜） |
| **対応SNS** | YouTube, Instagram等 |
| **技術難易度** | 低 |
| **メリット** | 総合SNS管理 |

## 検証計画

### Phase 1: ツール調査（3/2〜3/3）

- [ ] 各ツールの機能・価格詳細を調査
- [ ] 技術ドキュメント確認
- [ ] 無料トライアルの有無を確認

**調査開始**: 2026-03-02

### Phase 2: PoC実装（3/4）

**対象**: 最も有望なツール1〜2つ

**検証内容**:

| 検証項目 | 方法 | 判定基準 |
|----------|------|----------|
| **技術的実現性** | API連携テスト | 現在の実装パターンと親和性が高い |
| **データ精度** | サンプルクエリ実行 | Xリサーチと同等以上の有用性 |
| **レスポンス速度** | 応答時間計測 | ユーザー体験を損なわない（3秒以内） |
| **データ網羅性** | 複数キーワードでテスト | 主要情報が取得できる |

**サンプルクエリ**:
```
- 有名人AのYouTube出演動画
- トレンドBに関するInstagram投稿
- United Productions関連番組のSNS反応
```

### Phase 3: 評価・報告（3/5）

- [ ] 検証結果のまとめ
- [ ] 費用対効果の計算
- [ ] United Productions様へ報告
- [ ] 導入GO/NO-GO判断

## 判断基準

| 項目 | 基準 | 重み |
|------|------|------|
| **月額費用** | 〜5万円/月 | 高 |
| **技術的実現性** | 1週間以内に実装可能 | 高 |
| **データ精度** | Xリサーチと同等以上 | 高 |
| **レスポンス速度** | 3秒以内 | 中 |
| **運用工数** | 追加運用負荷が少ない | 中 |

## 期待される成果

- 有料ツール導入の判断材料
- PoC実装コード（再利用可能）
- 費用見積もり（月額・従量）

## 検証後のフロー

```
検証結果
    │
    ├─ GO（導入推奨）→ 費用見積もり → United Productions様承認 → 本格実装
    │
    └─ NO-GO（見送り）→ 理由を文書化 → 4月以降再検討 or 代替案検討
```

## リスクと対策

| リスク | 対策 |
|--------|------|
| 費用が予算を超える | 複数ツールを比較し、段階的導入を提案 |
| 技術的に困難 | 事前にドキュメントを精読し、PoCで確認 |
| 期間内に検証が終わらない | 3/3までに主要ツールの選定を完了 |

## メモ

- 現時点ではXリサーチを軸に据える
- 費用が高額な場合は見送りもあり得る
- 検証結果は3月5日の週次進捗報告会で共有
- 会議で「近いうちに費用はかかるもののYouTubeとInstagramの統合を試す」意向が示されている
