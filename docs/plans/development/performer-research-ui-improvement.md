## 目的

出演者リサーチ機能のUI/UXを改善し、現場で使いやすい形に整える。

## 現状の課題

### 1. 思考プロセスの表示

**問題**:
- AIの思考プロセス（Chain of Thought）が表示されてしまう
- ユーザーにとって不要な情報が表示される

**対応**:
- [ ] 思考プロセスを非表示にする
- [ ] または、折りたたみ表示にする

### 2. 自動スクロール

**問題**:
- リサーチ中に自動で下にスクロールされてしまう
- ユーザーが読んでいる途中で画面が動く

**対応**:
- [ ] 自動スクロールの制御
- [ ] ユーザーが手動でスクロールした場合は自動スクロールを停止

### 3. 表示速度

**問題**:
- リサーチ結果の表示に時間がかかる
- ストリーミング表示の最適化が必要

**対応**:
- [ ] レスポンスのストリーミング表示改善
- [ ] ローディング表示の最適化

## 改善項目

### UI調整

- [ ] 思考プロセスの非表示化
- [ ] 自動スクロール制御
- [ ] リサーチ進行状況の表示（プログレスインジケータ）
- [ ] 結果表示のレイアウト調整

### UX改善

- [ ] リサーチ中のキャンセル機能
- [ ] リサーチ結果のコピー機能
- [ ] リサーチ結果のエクスポート（将来的に）
