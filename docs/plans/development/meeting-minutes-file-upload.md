## 目的

議事録作成機能を「VTT/TXTファイルのアップロード」形式に変更し、現場のワークフローに最適化する。

## 背景

- 当初はテキストコピペも検討していたが、ファイルアップロードが最も効率的と判断
- Zoomから出力されるVTTまたはTXTファイルをそのまま使用

## 仕様

### 対応ファイル形式

| 形式 | 拡張子 | 優先度 |
|------|--------|--------|
| VTT | `.vtt` | 高 |
| TXT | `.txt` | 高 |

### VTTファイル仕様

Zoomが出力するVTT形式：
```
WEBVTT

00:00:00.000 --> 00:00:05.000
発言者A: こんにちは、会議を始めます。

00:00:05.000 --> 00:00:10.000
発言者B: はい、お願いします。
```

### 処理フロー

```
1. ユーザーがファイルを選択・アップロード
2. ファイル形式を検証
3. 内容をパース（VTT→テキスト抽出）
4. AIに送信し、議事録を生成
5. 整形された議事録を表示
```
