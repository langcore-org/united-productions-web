# 新規企画立案機能の移植

> **優先度**: 🔴 高  
> **期限**: 2026-03-12  
> **担当**: 浅野宏耀  
> **ステータス**: 未着手  
> **親タスク**: [実装プラン 2026年3月](../implementation-plan-2026-03.md)

---

## 目的

過去に開発した対話型新規企画立案機能を、現在のアプリに移植する。

## 背景

- 過去に対話型の新規企画立案機能を開発済み
- プロンプトを移植することで、現在のアプリに統合可能
- 過去の番組情報インプットなしで、まずは移植・動作確認を行う

## 実装方針

### 基本方針

```
過去アプリのプロンプト → 現在のアプリに移植 → 動作確認 → 調整
```

### 過去アプリからの移植要素

1. **対話フロー設計**
   - ユーザーとの対話パターン
   - 質問の順序・内容
   - 回答に応じたフォローアップ

2. **企画生成ロジック**
   - アイデア出しの手法
   - 企画の構成要素
   - 出力形式

3. **システムプロンプト**
   - AIの役割定義
   - 進行方法の指示
   - 出力ルール

## 仕様

### 機能名
- **正式名称**: 「新企画立案」（暫定）
- **サブテキスト**: 「アイデアを出してくれます」（案）

### 動作イメージ

```
1. ユーザー: 新しい番組企画を考えたい
2. AI: どのようなジャンルをお考えですか？
3. ユーザー: バラエティで...
4. AI: ターゲット層はどの世代を想定されていますか？
5. ...（対話を繰り返し）
6. AI: では、以下の企画案をご提案します...
```

### 出力内容

- 企画タイトル案
- コンセプト
- 進行イメージ
- 想定出演者（候補）
- ターゲット層
- 放映枠の提案
