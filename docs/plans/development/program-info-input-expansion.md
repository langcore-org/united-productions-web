## タスク内容

### 1. 正確な番組情報の収集

収集対象：
- [x] United Productions様のレギュラー番組一覧（正確なもの）
- [x] 各番組の基本情報
  - [x] 番組名
  - [x] 放送時間・曜日
  - [x] 放送局
  - [x] 番組概要
  - [x] レギュラー出演者
  - [x] 直近の放送回データ

### 2. データ形式の整備

```typescript
// 番組情報のデータ構造
interface ProgramInfo {
  id: string;
  name: string;           // 番組名
  broadcastTime: string;  // 放送時間
  broadcastDay: string;   // 放送曜日
  station: string;        // 放送局
  description: string;    // 番組概要
  regularCast: string[];  // レギュラー出演者
  genre: string;          // ジャンル
  recentEpisodes?: {      // 直近の放送回（追加）
    date: string;
    title: string;
    guests?: string[];
  }[];
}
```

2. **✅ データ構造化済み**
   - `lib/knowledge/programs.ts` に保存

3. **⏳ 最終確認・承認待ち**
   - 浅野による最終確認
   - UP側への確認・承認

## 品質管理プロセス（確立済み）

```
1. AIリサーチで情報収集
2. 複数ソースでのクロスチェック
3. コードレビュー（誤り検出）
4. 修正・無効化（必要に応じて）
5. 最終確認（浅野）
6. UP側確認・承認
```
