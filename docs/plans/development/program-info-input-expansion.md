## 目的

United Productions様のレギュラー番組情報を正確に収集・整備し、AIが参照できる形でインプットする。

## 現状

- AIによるリサーチで番組情報を取得済みだが、誤りが含まれている
- 正確な情報に差し替える必要がある

## タスク内容

### 1. 正確な番組情報の収集

収集対象：
- [ ] United Productions様のレギュラー番組一覧（正確なもの）
- [ ] 各番組の基本情報
  - 番組名
  - 放送時間・曜日
  - 放送局
  - 番組概要
  - レギュラー出演者

### 2. データ形式の整備

```typescript
// 番組情報のデータ構造
interface ProgramInfo {
  id: string;
  name: string;           // 番組名
  broadcastTime: string;  // 放送時間
  broadcastDay: string;   // 放送曜日
  station: string;        // 放送局
  description: string;    // 番組概要
  regularCast: string[];  // レギュラー出演者
  genre: string;          // ジャンル
}
```

### 3. データベースへの登録

- [ ] 既存データの修正
- [ ] 不足データの追加
- [ ] 検証（AIが正しく参照できるか）

## 収集方法

1. **United Productions様から情報提供を依頼**
   - レギュラー番組一覧の正確なリスト
   - 各番組の詳細情報

2. **AIが参照しやすい形式に変換**
   - 構造化データとして保存
   - 検索・参照が高速に行えるようインデックス作成

## 影響範囲

この情報が使用される機能：
- 出演者リサーチ
- エビデンスリサーチ
- 新企画立案
- チャット（番組選択時）
