# 社内発表用スクリーンショット作成

> **優先度**: 🟡 中  
> **期限**: 2026-03-02（来月第1月曜日まで）  
> **担当**: 浅野宏耀  
> **ステータス**: 未着手  
> **親タスク**: [実装プラン 2026年3月](../implementation-plan-2026-03.md)

---

## 目的

来月第1月曜日の社内朝礼での発表用に、アプリのスクリーンショットを準備する。

## 要件

### 発表内容

島田源太郎が社内朝礼で以下を発表：
- 3月末にローンチ予定であること
- リリース機能：リサーチ、議事録作成、新規企画立案
- **ナレーション原稿作成は省いたラインナップ**

### 必要なスクリーンショット

| 用途 | 内容 | 形式 |
|------|------|------|
| **メイン** | チャット画面（空の状態） | PNG/JPG |
| **機能紹介** | サイドバー表示（全機能アイコン） | PNG/JPG |
| **補足** | 各機能の簡単なキャプチャ（必要に応じて） | PNG/JPG |

## スクリーンショット仕様

### メイン画面（チャット）

```
【表示内容】
- チャットインターフェース
- サイドバー（全機能が見える状態）
- 空のチャットエリア
- 入力欄

【注意点】
- ナレーション原稿作成アイコンは非表示
- 清潔感のある状態
- 機密情報が映っていないこと
```

### サイドバー表示

```
【表示する機能アイコン】
1. チャット
2. 出演者リサーチ
3. エビデンスリサーチ
4. 議事録作成
5. 新企画立案
6. 履歴

【非表示】
- ナレーション原稿作成
```
