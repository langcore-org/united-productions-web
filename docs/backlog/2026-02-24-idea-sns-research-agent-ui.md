```
┌─────────────────────────────────────────┐  ← カード全体（角丸8px、影付き）
│                                         │
```

## 3. インタラクション

### 3.1 基本操作

| 操作 | 動作 |
|------|------|
| **カードクリック** | 折りたたみ/展開の切り替え |
| **展開ボタン▼** | すべて展開 / すべて折りたたみ |
| **ログコピー** | クリップボードにコピー |
| **特定ステップ再実行** | 該当ステップから再開 |

### 3.2 自動動作

| タイミング | 動作 |
|-----------|------|
| **新規ステップ開始** | 自動的にカード追加 + 展開 |
| **ステップ完了** | アイコン変更 + 完了表示 + 自動折りたたみ（オプション） |
| **エラー発生** | エラーカードを展開 + スクロールで表示 |
| **全体完了** | 完了サマリーカードを最上部に固定表示 |

### 3.3 表示要素

| 要素 | 表示内容 |
|------|---------|
| **タイムスタンプ** | `HH:MM:SS` 形式 |
| **実行時間** | `(Xs)` または `(Xm Ys)` 形式 |
| **プログレスバー** | パーセンテージ表示付き |
| **詳細ログ** | インデント付きテキスト |
| **サブステップ** | ネストされたカード |

## 5. アニメーション仕様

### 5.1 パルス（点滅）アニメーション

```css
/* 実行中カードの背景 */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}

.card-running {
  animation: pulse 2s infinite;
  border-color: #3b82f6; /* 青色 */
  background-color: rgba(59, 130, 246, 0.05);
}
```

### 5.2 プログレスバー

```
初期状態: ░░░░░░░░░░ 0%
進行中:   ▓▓▓▓▓░░░░░ 50%
完了:     ▓▓▓▓▓▓▓▓▓▓ 100%
```

- アニメーション: スムーズな幅の遷移（transition: width 0.3s ease）
- 色: 青（実行中）→ 緑（完了）→ 赤（エラー）

### 5.3 カード追加アニメーション

```css
/* 新規カード追加時 */
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateY(-10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.card-new {
  animation: slideIn 0.3s ease-out;
}
```

### 5.4 自動スクロール

- 新規カード追加時: 自動的に最下部へスクロール
- ユーザーが手動スクロールした場合: 自動スクロールを一時停止
- 完了時: 完了サマリーカードへスクロール
