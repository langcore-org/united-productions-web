# AI Hub 社内リリースのご案内

> **テレビ制作業務をAIで効率化する新ツール「AI Hub」をリリースしました**
> 
> **リリース日**: 2026年3月26日

---

## 📢 リリース概要

日頃の制作業務の効率化を目的に、AI支援ツール「AI Hub」を社内リリースしました。

本システムは議事録作成、出演者リサーチ、エビデンス（情報）検証、新企画立案など、制作業務の様々な場面でAIがサポートします。


---

## 🚀 ご利用開始方法

### アクセスURL

```
https://united-productions-web.vercel.app/
```

### ログイン方法

1. 上記URLにアクセス
2. 「Googleでログイン」をクリック
3. **会社配布のGoogleアカウント**でログイン

> ⚠️ **注意**: 個人のGoogleアカウントではアクセスできません。必ず会社配布のアカウントをご使用ください。

## 📋 提供機能一覧

サイドバーから以下5機能が利用できます。

| 機能 | 用途 | 工数削減例 |
|-----|------|-----------|
| **💬 チャット** | 一般的な質問や相談、最新情報の調査 | 調査半日 → **5分** |
| **👥 出演者リサーチ** | 企画に合った出演者候補をAIが提案 | 半日〜1日 → **15分** |
| **🛡️ エビデンスリサーチ** | 情報の真偽をAIが検証 | 半日〜1日 → **15分** |
| **📝 議事録作成** | 会議の文字起こしを整形 | 1時間 → **5分** |
| **💡 新企画立案** | 番組情報から企画案を生成 | 半日 → **30分** |

## 📝 基本的な使い方

### 操作フロー（全機能共通）

```
1. サイドバーから機能を選択
   ↓
2. 指示やテキストを入力欄に入れる
   ↓
3. 送信ボタンをクリック
   ↓
4. AIの応答を待つ（数秒〜数十秒）
   ↓
5. 結果を確認・コピー・保存
```

### 活用例

#### 出演者リサーチ
```
【入力例】
プログラミング教育番組の企画です。
30代で、子供向けに分かりやすく説明できる出演者を探しています。

【結果】
・候補者リスト（30人程度）
・各候補のプロフィール・出演実績
・番組との相性分析
```

#### 議事録作成
```
【手順】
1. Zoomで「文字起こし」をダウンロード
2. AI Hubで「議事録作成」を開く
3. 文字起こしテキストを貼り付け
4. 「会議用」または「面談用」を選択
5. 送信 → 整形された議事録が完成
```
