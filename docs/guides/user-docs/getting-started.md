# はじめてのAI Hub

> **初めてAI Hubを使う方へのガイド**
> 
> **最終更新**: 2026-03-20 15:47

## AI Hubとは

テレビ制作業務を支援するAIツールです。以下の業務を効率化します：

| 機能 | できること | 従来の工数 |
|-----|-----------|-----------|
| **チャット** | 一般的な質問や相談、最新情報の調査 | 調査半日 → **5分** |
| **出演者リサーチ** | 企画に合った出演者候補をAIが提案 | 半日〜1日 → **15分** |
| **エビデンスリサーチ** | 情報の真偽をAIが検証 | 半日〜1日 → **15分** |
| **議事録作成** | 会議の文字起こしを整形 | 1時間 → **5分** |
| **新企画立案** | 番組情報から企画案を生成 | 半日 → **30分** |

## 始める前に

### 必要なもの

- Googleアカウント（会社配布のもの）
- インターネット接続
- 推奨ブラウザ: Chrome, Safari, Edge

### アクセス方法

1. ブラウザで `https://up-agent.vercel.app` を開く
2. 「Googleでログイン」をクリック
3. 会社のGoogleアカウントでログイン

> **認証について**: 本システムはSupabase Authを使用してGoogle OAuthでログインします。

## 基本的な使い方

### 1. 画面の構成

```
┌─────────────────────────────────────┐
│  ロゴ    検索...          👤 ユーザー  │  ← ヘッダー
├────────┬────────────────────────────┤
│        │                            │
│ リサーチ │                            │
│ ├出演者  │     メインコンテンツ        │
│ ├エビデンス│     （チャット画面等）       │
│ │       │                            │
│ 議事録   │                            │
│ 新企画   │                            │
│ 番組設定  │                            │
│        │                            │
└────────┴────────────────────────────┘
     ↑
   サイドバー（機能メニュー）
```

### 2. 各機能の使い方（共通）

ほとんどの機能は同じ操作フローです：

```
1. サイドバーから機能を選択
   ↓
2. 指示やテキストを入力欄に入れる
   ↓
3. 送信ボタンをクリック
   ↓
4. AIの応答を待つ（数秒〜数十秒）
   ↓
5. 結果を確認・コピー・保存
```

## 具体的な使用例

### 例1: 出演者リサーチ

**状況**: 「プログラミングに詳しい30代の出演者を探したい」

1. サイドバー「出演者リサーチ」をクリック
2. 入力欄に以下を入力：
   ```
   プログラミング教育番組の企画です。
   30代で、子供向けに分かりやすく説明できる出演者を探しています。
   ```
3. 送信ボタンをクリック
4. AIが候補者リスト（30人程度）を提示
5. 気になる候補者をメモまたはコピー

### 例2: 議事録作成

**状況**: 「Zoom会議の文字起こしを議事録にしたい」

1. Zoomで「文字起こし」をダウンロード
2. AI Hubで「議事録作成」を開く
3. 文字起こしテキストを貼り付け
4. 「会議用」または「面談用」を選択
5. 送信 → 整形された議事録が完成
