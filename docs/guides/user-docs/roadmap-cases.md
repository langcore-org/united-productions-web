## 活用事例

### 事例1: 新規企画の出演者発掘

**制作**: 「おもしろ科学実験」企画チーム

**課題**:
- 子供に科学を分かりやすく教えられる出演者を探す必要
- 従来は制作会社に依頼（1件¥50,000、1週間かかる）

**AI Hubの活用**:
```
入力: 「小学生向け科学実験番組。子供に人気で、
       実験が上手い先生またはYouTuberを探したい。
       関東圏在住で収録に来られる方」
```

**結果**:
- 候補者30名を15分で取得
- その中から3名に絞り、即日アプローチ
- **工数: 1週間 → 半日**
- **コスト: ¥50,000 → ¥10**

---

### 事例2: 緊急の企画変更対応

**制作**: 「路線バス旅」ロケチーム

**課題**:
- 雨天により屋外ロケが不可能に
- 急遽、屋内企画に変更する必要

**AI Hubの活用**:
```
入力: 「〇〇駅周辺で雨天でも撮影できる屋内スポット。
       観光客が少なく、撮影許可が取りやすい場所」
```

**結果**:
- 候補5件を10分で取得
- 電話確認の上、1件に決定
- **対応時間: 2時間 → 30分**

---

### 事例3: 議事録作成の効率化

**制作**: 制作部全般（週4回の定例會議）

**課題**:
- 打ち合わせ後、議事録作成に1時間かかる
- 翌日までに共有されるため、確認が遅れる

**AI Hubの活用**:
- Zoomの文字起こしをそのまま貼り付け
- 5分で整形された議事録が完成

**結果**:
- **議事録作成: 1時間 → 5分**
- **共有タイミング: 翌日 → 即日**
- 月間工数削減: **4時間 → 20分**

---

### 事例4: エビデンスリサーチでの情報検証

**制作**: 「情報番組」調査チーム

**課題**:
- VTR素材の情報正確性を確認するのに時間がかかる
- 専門的な主張の真偽を調べるのに手間がかかる

**AI Hubの活用**:
```
入力: 「AIは人間の仕事を奪うという主張について、
       専門家の見解と統計データを交えて検証してください」
```

**結果**:
- 出典付きの検証結果を15分で取得
- 複数の専門家見解をまとめて提示
- **情報検証: 2時間 → 15分**
- 放送で使用できる信頼性の高い情報として活用

## 活用のコツ（ベストプラクティス）

### 入力のコツ

| 良い例 | 効果 |
|-------|------|
| 「30代、女性、料理が得意」 | 具体的な条件で精度向上 |
| 「東京駅周辺、徒歩5分圏内」 | 範囲を絞って無駄を減らす |
| 「〜を除く」「〜は不要」 | 除外条件でノイズ削減 |
| 「過去に〇〇で出演経験がある人」 | 実績条件で質向上 |

### 業務への組み込み方

1. **最初は小さく始める**
   - 1機能だけ試す（例: 議事録作成のみ）
   - 慣れてから他機能も活用

2. **チーム内で役割分担**
   - リサーチ担当: AI Hubで事前調査
   - 演出: 結果を基に本調査・交渉

3. **結果の保存・共有**
   - 良い結果はメモ帳等に保存
   - チーム内で共有（将来的にはシステム内で共有可能に）
