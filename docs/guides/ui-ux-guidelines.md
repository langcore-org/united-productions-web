# AI Hub UI改善ドキュメント

## 1. 概要

### 1.1 目的

AI HubプロジェクトのUIをGrok/Kimi風のモダンなデザインに刷新し、以下の目標を達成すること：

- **視覚的な一貫性**: 全ページで統一されたデザイン言語を実現
- **操作性の向上**: 直感的で使いやすいインターフェースを提供
- **ブランドイメージの強化**: プロフェッショナルで洗練された印象を与える
- **ダークモードの最適化**: 制作業務に適した暗色テーマを採用

## 2. デザイン方針

### 2.1 Grok/Kimi風デザインの特徴

Grok（xAI）とKimi（Moonshot AI）のデザイン哲学を参考に、以下の要素を取り入れました：

#### 視覚的特徴
- **ダークテーマ**: 深い黒（#0d0d12, #0a0a0f）を基調とした背景
- **アクセントカラー**: 鮮やかなオレンジ（#ff6b00）をブランドカラーとして使用
- **ガラスモーフィズム**: backdrop-blur効果による半透明の重ね合わせ
- **微妙な境界線**: 低コントラストのボーダー（#2a2a35）で要素を区切る

#### インタラクション設計
- **ホバーエフェクト**: スケール変化と色の遷移によるフィードバック
- **スムーズなアニメーション**: 200-300msのトランジションで自然な動き
- **フォーカス状態**: リング効果とボーダー色の変化で明示

### 2.2 デザイン原則

```
1. シンプリシティ: 不要な装飾を排除し、コンテンツを優先
2. 階層性: サイズ、色、スペースで視覚的階層を構築
3. 一貫性: 同じパターンを全ページで統一的に使用
4. フィードバック: ユーザーのアクションに対する明確な応答
```

### 3.1 Sidebar

**ファイル**: `components/layout/Sidebar.tsx`

#### 主な変更点

| 項目 | 変更前 | 変更後 |
|-----|-------|-------|
| 背景色 | #1a1a24 | #0d0d12 |
| ボーダー | #2a2a35 | #2a2a35/60（半透明）|
| アクティブ状態 | 単色背景 | グラデーション + 左側インジケーター |

- **折りたたみ機能**: クリックでサイドバーを最小化/展開
- **履歴セクション**: 時系列でグループ化された履歴表示
- **ホバーアクション**: 履歴項目の編集・削除ボタン
- **カスタムスクロールバー**: ダークテーマに適した細いスクロールバー

```typescript

// テキスト
text-white          // 主要テキスト
text-gray-200       // セカンダリテキスト
text-gray-400       // 非アクティブテキスト

// アクセント
text-[#ff6b00]      // アクティブ/ブランドカラー
bg-[#ff6b00]        // ボタン/インジケーター
```
