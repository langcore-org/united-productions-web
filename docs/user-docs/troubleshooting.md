## ログイン・認証

### ログインできない

| 症状 | 原因 | 解決方法 |
|-----|------|---------|
| 「アクセス権がありません」と表示 | 許可されていないGoogleアカウント | 会社配布のGoogleアカウントでログイン |
| ログイン画面がループする | ブラウザのキャッシュ問題 | Chromeの場合: Ctrl+Shift+Delete でキャッシュ削除 |
| 「セッションが切れました」 | 長時間放置 | 再度ログインし直す |

### 画面が真っ白になる

1. ブラウザを更新（F5または⌘+R）
2. 改善しない場合は、シークレットウィンドウで開く
3. それでも改善しない場合は、管理者に連絡

## 機能の使用

### AIの応答がない/遅い

| 状況 | 対応 |
|-----|------|
| 30秒以上経っても応答なし | ① ページを更新 ② もう一度送信 ③ 改善しない場合は別の機能を試す |
| 「エラーが発生しました」 | ① 入力内容を確認 ② 短くして再送信 ③ 改善しない場合は管理者に連絡 |
| 応答が途中で止まる | ① ネットワーク接続を確認 ② 再送信 ③ 長い入力は分割して送信 |

### 入力した内容が消えた

- **原因**: ページ遷移や更新で入力中の内容が失われる
- **対策**: 長い入力はメモ帳等にコピーしてから貼り付け
- **復元**: ブラウザの戻るボタンで戻ると復元される場合あり

### 結果のコピーがうまくいかない

| 状況 | 対応 |
|-----|------|
| コピーボタンが反応しない | 手動でテキストを選択 → Ctrl+C（⌘+C） |
| Wordに貼り付けると崩れる | 「貼り付けのオプション」→「テキストのみ保持」を選択 |
| 特殊文字が文字化け | フォントを「MSゴシック」または「Noto Sans JP」に変更 |

## 各機能の問題

### リサーチ機能

| 症状 | 解決方法 |
|-----|---------|
| 検索結果が期待と違う | 入力をより具体的に（年齢、職業、特徴等） |
| 古い情報が出る | 「2024年以降の情報で」等、期間を指定 |
| 候補が少ない | 条件を緩くする（「〜であれば尚可」等） |

### 議事録作成

| 症状 | 解決方法 |
|-----|---------|
| 話者が区別されない | Zoomの文字起こし設定で「話者名」を有効化 |
| 専門用語が変換される | 結果を手動で修正、または「以下の専門用語をそのまま使用」で指定 |
| 長すぎて処理されない | 文字起こしを分割して送信（30分単位程度） |

### NA原稿作成

| 症状 | 解決方法 |
|-----|---------|
| 話者ラベルがおかしい | 手動で修正（AIの推測なので100%ではない） |
| 「えー」「あー」が残る | 「フィラーワードを除去」等、指示に追加 |
| 1行が長すぎる | 「1行は30文字以内で改行」を指示に追加 |

## パフォーマンス

### 画面の表示が遅い

1. インターネット接続を確認
2. ブラウザのタブを減らす
3. ブラウザを再起動
4. それでも改善しない場合は、別のブラウザを試す（Chrome推奨）

### スマートフォンで使いにくい

- **現状**: デスクトップ・タブレット推奨
- **対応**: 基本的な操作は可能だが、入力欄が小さい場合あり
- **推奨**: 資料作成等はPCを使用

## 自己診断チェックリスト

問題が発生したら、まず以下を確認:

- [ ] インターネット接続は正常か？
- [ ] ブラウザは最新版か？
- [ ] 会社のGoogleアカウントでログインしているか？
- [ ] ページを更新（F5）しても改善しないか？
- [ ] シークレットウィンドウでも同じ問題が起きるか？

上記を確認しても解決しない場合は、管理者に連絡。
