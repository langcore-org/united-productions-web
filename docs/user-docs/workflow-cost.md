# 業務統合・コスト管理ガイド

> **制作現場への組み込み方とコスト管理**
> 
> **最終更新**: 2026-02-20 13:15

## 業務フローへの組み込み

### 企画段階

```
【従来のフロー】
企画会議 → アイデア出し（半日） → リサーチ（1-2日） → 企画書作成（1日）
                                    ↑
                              出演者探し等

【AI Hub導入後】
企画会議 → AIでリサーチ並行（15分） → 企画書作成（1日）
         ↓
    新企画立案機能でアイデア補助
```

**効果:** 企画段階のリサーチ工数を**1-2日 → 数時間**に短縮

---

### リサーチ段階

```
【従来のフロー】
演出がリサーチ → 情報整理 → スタッフに共有（半日〜1日）

【AI Hub導入後】
演出がAIに指示 → AIがリサーチ・整理 → そのまま共有（15分）
```

**効果:** リサーチ工数を**半日〜1日 → 15分**に短縮

---

### 打ち合わせ段階

```
【従来のフロー】
打ち合わせ → 議事録作成（1時間） → 共有 → 確認（翌日）

【AI Hub導入後】
打ち合わせ → 議事録自動作成（5分） → 即時共有
```

**効果:** 議事録作成を**1時間 → 5分**に短縮

---

### 編集段階

```
【従来のフロー】
書き起こし → NA原稿作成（2時間） → ナレーター収録

【AI Hub導入後】
書き起こし → NA原稿自動作成（10分） → ナレーター収録
```

**効果:** NA原稿作成を**2時間 → 10分**に短縮

---

## 導入効果（工数削減例）

### 1本の番組制作における効果（例）

| 業務 | 従来工数 | AI Hub使用時 | 削減工数 |
|-----|---------|-------------|---------|
| 出演者リサーチ | 8時間 | 1時間 | **7時間** |
| 企画書リサーチ | 8時間 | 2時間 | **6時間** |
| 議事録作成（週2回×4週間） | 8時間 | 1時間 | **7時間** |
| NA原稿作成 | 8時間 | 1時間 | **7時間** |

**1本あたり約34時間（4日分以上）の工数削減**

## コスト管理

### 料金体系

| 項目 | 料金 | 備考 |
|-----|------|------|
| AI Hub利用料 | **無料** | システム使用料は無料 |
| LLM API使用料 | 従量課金 | 実際のAI使用分のみ |
| インフラ費用 | **無料** | Vercel/Neon/Upstash無料枠内 |

### LLM API使用料目安

| 使用パターン | 月額目安 |
|-----------|---------|
| 軽度（週1-2回） | **¥0**（無料枠内） |
| 標準（週3-5回） | **¥100〜300** |
| 活発（毎日使用） | **¥500〜1,000** |
| チーム全員（10人） | **¥2,000〜5,000** |

### コスト削減のコツ

1. **Geminiを優先使用**
   - Google AI Studio無料枠: 月1,500リクエスト
   - 議事録、NA原稿等はGeminiで十分

2. **Grok/Perplexityは必要時のみ**
   - リアルタイム情報（X検索）が必要な場合のみ使用
   - 通常のリサーチはGeminiで代替可能

3. **キャッシュの活用**
   - 同じ質問は24時間キャッシュされる
   - 再検索時はコストゼロ
