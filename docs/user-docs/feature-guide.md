# 機能ガイド

> **AI Hubの全機能と操作手順**
> 
> **最終更新**: 2026-02-20 13:15

## 機能一覧

| 機能 | 用途 | 入力 | 出力 |
|-----|------|------|------|
| **出演者リサーチ** | 企画に合った出演者候補を提案 | 企画内容 | 候補者リスト（30人） |
| **場所リサーチ** | ロケ地候補と撮影条件を調査 | ロケのテーマ | 場所リスト・撮影ポイント |
| **情報リサーチ** | テーマに関する情報を収集 | 調査テーマ | 整理された情報 |
| **エビデンスリサーチ** | 情報の真偽を検証 | 検証したい情報 | 検証結果・出典 |
| **議事録作成** | 会議の文字起こしを整形 | Zoom文字起こし | 整形済み議事録 |
| **NA原稿作成** | 書き起こしをナレーション原稿に | Premiere書き起こし | NA原稿（Word貼り付け対応） |
| **新企画立案** | 番組情報から企画案を生成 | 番組設定 | 企画案・演出案 |

## リサーチ機能（4機能）

### 出演者リサーチ

**使い方:**
1. サイドバー「リサーチ」→「出演者リサーチ」をクリック
2. 企画の内容を入力
3. 送信

**入力例:**
```
タイトル: 子供向けプログラミング教室
内容: 小学生が初めてプログラミングに触れる番組
求める出演者: 子供に分かりやすく教えられる、親しみやすい方
```

**出力例:**
```
## 候補者リスト

1. **山田太郎**（32歳）
   - 職業: プログラマー / YouTuber
   - 特徴: 子供向けプログラミング書の著者
   - 出演実績: NHK教育番組

2. ...（計30人）
```

**活用ポイント:**
- 具体的な条件（年齢、職業、特徴）を入れると精度が上がる
- 「〜を除く」「〜優先」などの条件も伝えると良い

---

### 場所リサーチ

**使い方:**
1. サイドバー「リサーチ」→「場所リサーチ」をクリック
2. ロケのテーマ・条件を入力
3. 送信

**入力例:**
```
テーマ: 東京近郊で初心者でも楽しめるサーフィンスポット
撮影時期: 9月
求める条件: 駐車場あり、レンタル器材あり
```

**活用ポイント:**
- 撮影時期を入れると、その時期の状況（混雑度、波の状況等）も調査
- 「過去に〜で撮影したことがある」等の情報も共有可能

---

### 情報リサーチ

**使い方:**
1. サイドバー「リサーチ」→「情報リサーチ」をクリック
2. 調査したいテーマを入力
3. 送信

**活用ポイント:**
- 広いテーマでも、具体的な角度を指定すると良い
- 「視聴者に伝えたいポイント」を入れると整理しやすい

---

### エビデンスリサーチ

**使い方:**
1. サイドバー「リサーチ」→「エビデンスリサーチ」をクリック
2. 検証したい情報を入力
3. 送信

**活用ポイント:**
- 「本当？」と思った情報をそのまま貼り付けて検証
- 出典付きで返答されるので、放送で使える

## 制作支援機能（3機能）

### 議事録作成

**使い方:**
1. Zoomで会議の文字起こしをダウンロード
   - Zoom → 「録画」→ 該当会議 → 「文字起こし」→ ダウンロード
2. AI Hubで「議事録作成」を開く
3. 文字起こしテキストを貼り付け
4. 「会議用」または「面談用」を選択
5. 送信

**テンプレート:**
- **会議用**: 議題、発言要旨、決定事項、TODO
- **面談用**: 人物名、経歴、話した内容、出演可否

**出力形式:**
```markdown
## 会議概要
- 日時: 2026/02/20 14:00-15:30
- 参加者: 山田、佐藤、鈴木

## 議題1: 次回ロケについて
### 発言要旨
- 山田: 天気予報を確認したところ、晴れの見込み
- 佐藤: 出演者のスケジュールは調整済み

### 決定事項
- ロケ日: 3月15日
- 集合場所: 現地直接

### TODO
- [ ] 山田: 車両手配（2/25まで）
- [ ] 佐藤: 出演者に最終確認
```

### NA原稿作成

**使い方:**
1. Premiere Proで書き起こしテキストをコピー
2. AI Hubで「文字起こし」→「NA原稿作成」を開く
3. テキストを貼り付け
4. 送信

**特徴:**
- 話者ラベル（櫻井、末澤等）を自動付与
- 不要な言い淀みを除去
- **Wordへの貼り付けに最適化**された形式で出力

---

### 新企画立案

**使い方:**
1. 「番組設定」で番組情報を登録（初回のみ）
2. 「新企画立案」を開く
3. 企画の方向性を入力
4. 送信

**入力例:**
```
今回のテーマ: 地方の伝統工芸に注目したい
ターゲット: 30-40代女性
求めるもの: 若い職人にスポットを当てた企画案
```

**活用ポイント:**
- 番組設定（コンセプト、過去企画）を充実させると精度が上がる
- 「〜っぽい企画」「〜を避けたい」等の要望も伝えると良い
