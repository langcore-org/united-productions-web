# 機能ガイド

> **Teddyの全機能と操作手順**
> 
> **最終更新**: 2026-03-12 10:30

## 機能一覧

現在、サイドバーから利用できる機能は以下の5つです。

| 機能 | 用途 | 入力 | 番組選択 | ファイル添付 |
|-----|------|------|---------|-------------|
| **チャット** | 一般的な質問や相談 | 自由形式 | 不要 | ✅ |
| **出演者リサーチ** | 企画に適した出演者候補を提案 | 企画内容・テーマ | **必須** | - |
| **エビデンスリサーチ** | 情報の真偽を検証 | 検証したい情報・主張 | 不要 | - |
| **議事録作成** | 文字起こしから議事録を作成 | 文字起こしテキスト | 不要 | **対応** |
| **新企画立案** | 番組情報から企画案を生成 | 企画の方向性・テーマ | **必須** | - |

> **未実装機能（将来追加予定）**: 情報リサーチ（テーマに関する情報を収集）はURL直接アクセスでのみ利用可能です。

## チャット機能（5機能）

### チャット

**使い方:**
1. サイドバー「チャット」をクリック
2. 質問や相談内容を入力
3. 送信

**できること:**
- 一般的な質問への回答
- アイデア出しの支援
- 文章の推敲・校正
- 情報の整理・要約
- 制作業務に関する相談
- 最新情報の調査（Web検索）
- トレンド情報の収集（X検索）

**活用ポイント:**
- 何でも気軽に質問できます
- 最新情報が必要な場合は、自動的にWeb検索/X検索が実行されます

---

### 出演者リサーチ

**使い方:**
1. サイドバー「出演者リサーチ」をクリック
2. 番組を選択（必須）
3. 企画の内容・テーマを入力
4. 送信

**入力例:**
```
テーマ: 子供向けプログラミング教室
内容: 小学生が初めてプログラミングに触れる番組
求める出演者: 子供に分かりやすく教えられる、親しみやすい方
```

**プロンプトサジェスト（自動表示）:**
- 「最も出演可能性が高い候補3名について、具体的なプレゼン企画書を作成してください」
- 「これらの候補者の中から、番組プロデューサーへの売り込み用のピッチ資料（スライド形式）を作ってください」
- 「過去の高視聴率を記録した回の傾向を分析して、成功パターンを表にまとめてください」
- 「実在する候補者を具体的にリサーチして、連絡先や実績とともにリスト化してください」

**活用ポイント:**
- 番組を選択することで、その番組の特性に合わせた出演者マッチングが行われます
- 具体的な条件（年齢、職業、特徴）を入れると精度が上がります
- Web検索とX検索を組み合わせて、実在する人物の情報も収集します

### エビデンスリサーチ

**使い方:**
1. サイドバー「エビデンスリサーチ」をクリック
2. 検証したい情報・主張を入力
3. 送信

**活用ポイント:**
- 「本当？」と思った情報をそのまま貼り付けて検証できます
- 出典付きで返答されるので、放送で使えます
- 反対意見や専門家の見解も調査可能です

**プロンプトサジェスト（自動表示）:**
- 「この主張の信憑性をさらに深く検証してください」
- 「反対の意見や反論も調査して提示してください」
- 「専門家の見解や学術的な根拠を追加で調べてください」

---

### 議事録作成

**使い方:**
1. Zoom等で会議の文字起こしを取得
2. サイドバー「議事録作成」をクリック
3. 文字起こしテキストを貼り付ける、またはファイルを添付
4. 送信

**対応ファイル形式:**
- テキストファイル（.txt）
- PDFファイル（.pdf）
- Wordファイル（.doc, .docx）
- 直接テキスト貼り付けも可能

**出力形式:**
```markdown
## 会議概要
- 日時: 2026/02/20 14:00-15:30
- 参加者: 山田、佐藤、鈴木

## 議題1: 次回ロケについて
### 発言要旨
- 山田: 天気予報を確認したところ、晴れの見込み
- 佐藤: 出演者のスケジュールは調整済み

### 決定事項
- ロケ日: 3月15日
- 集合場所: 現地直接

### TODO
- [ ] 山田: 車両手配（2/25まで）
- [ ] 佐藤: 出演者に最終確認
```

**プロンプトサジェスト（自動表示）:**
- 「議事録をMarkdown形式で整形してください」
- 「ToDoリストだけを抽出してください」
- 「決定事項を箇条書きでまとめてください」

### 新企画立案

**使い方:**
1. サイドバー「新企画立案」をクリック
2. 番組を選択（必須）
3. 企画の方向性・テーマ・条件を入力
4. 送信

**入力例:**
```
今回のテーマ: 地方の伝統工芸に注目したい
ターゲット: 30-40代女性
求めるもの: 若い職人にスポットを当てた企画案
条件: 感動系、ロケ企画
```

**プロンプトサジェスト（自動表示）:**
- 「この企画をより具体的に詳細化してください」
- 「予算とスケジュール案も追加してください」
- 「類似企画との差別化ポイントを強調してください」

**活用ポイント:**
- 番組を選択することで、その番組の特性に合わせた企画提案が行われます
- 「〜っぽい企画」「〜を避けたい」等の要望も伝えると良いです
- Web検索で類似企画やトレンドを調査し、差別化ポイントを提案します

## 技術的な仕組み

### 使用されるAIツール

全機能で以下のツールが自動的に使用される場合があります：

| ツール名 | 用途 |
|---------|------|
| **Web検索** | 最新情報・事実確認 |
| **X検索** | トレンド・世論の収集 |
| **コード実行** | データ処理・計算・テキスト整形 |

### 会話の記憶と要約

- 長い会話では、自動的に過去の会話が要約されます
- トークン数が10万を超えると、ClientMemoryが要約を実行
- 要約は「これまでの会話の要約：」としてシステムに保持されます

### フォローアップ質問

- AIの回答後、自動的に関連するフォローアップ質問が生成されます
- 生成には数秒かかる場合があります
- 表示された質問をクリックすると、そのまま続けて質問できます
