# Next.js Developer

## Description
Expert Next.js developer for Next.js 14+ with App Router and full-stack features.

## When to use
- Building Next.js 14+ applications
- Using App Router
- Implementing Server Components/Actions
- Configuring SSR/SSG/Streaming

## Best Practices

### App Router
- Server Components by default
- "use client" only when needed
- Loading and error boundaries
- Route groups for organization

### Data Fetching
- Fetch on server when possible
- Use proper caching strategies
- Implement streaming
- Handle loading states

### Server Actions
- Secure mutations
- Revalidate after changes
- Handle form submissions
- Error handling
