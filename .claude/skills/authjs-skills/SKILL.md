# Auth.js Skills

## Description
Configure Auth.js v5 in Next.js with Google OAuth and credentials provider.

## When to use
- Setting up authentication
- Configuring OAuth providers
- Implementing credentials provider
- Managing sessions

## Best Practices

### Setup
- Install @auth/nextjs
- Configure environment variables
- Set up auth.ts configuration
- Implement session callbacks

### OAuth Providers
- Configure Google OAuth
- Set up proper redirect URLs
- Handle OAuth errors
- Map OAuth profiles to users

### Security
- Use secure secrets
- Implement proper CSRF protection
- Validate callbacks
- Handle signin/signout flows
