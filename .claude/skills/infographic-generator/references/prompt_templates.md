# インフォグラフィック プロンプトテンプレート集

## 基本テンプレート

### データ統計型
```
Professional infographic about [TOPIC]:
Key statistics:
- [Stat 1]: [Value]
- [Stat 2]: [Value]
- [Stat 3]: [Value]

Style: Modern data visualization, clean lines, [COLOR SCHEME] color palette
Layout: Vertical with numbered sections
```

### トレンド分析型
```
Infographic showing [TREND/TOPIC] evolution:
Timeline: [YEAR START] to [YEAR END]
Key milestones:
- [Year]: [Event/Change]
- [Year]: [Event/Change]
- [Year]: [Event/Change]

Style: Timeline-based design, gradient progression, professional icons
```

### 比較型
```
Comparison infographic: [ITEM A] vs [ITEM B]
Categories:
- [Category 1]: A=[Value], B=[Value]
- [Category 2]: A=[Value], B=[Value]
- [Category 3]: A=[Value], B=[Value]

Style: Split design, contrasting colors, clear visual comparison
```

### プロセス・フロー型
```
Step-by-step infographic for [PROCESS]:
Steps:
1. [Step name]: [Brief description]
2. [Step name]: [Brief description]
3. [Step name]: [Brief description]

Style: Flowchart design, numbered icons, connecting arrows
```

## カラースキーム推奨

### ビジネス・経済
- Blue and white (信頼性)
- Navy and gold (高級感)
- Green and white (成長)

### テクノロジー
- Blue and purple gradient (未来的)
- Cyan and dark background (テック感)
- Neon accents on dark (サイバー)

### エンターテイメント
- Vibrant multi-color (楽しさ)
- Red and black (インパクト)
- Pink and purple (ポップ)

### 健康・ライフスタイル
- Green and natural tones (自然)
- Light blue and white (清潔感)
- Warm earth tones (オーガニック)

## 使用例

### 例1: AIゲーム市場
```
Professional infographic about AI in Gaming Industry 2024:
Market size: $50 billion (projected 2030)
Current applications:
- NPC Behavior: 35% of games
- Procedural Generation: 28% adoption
- Player Personalization: 22% implementation
Growth rate: 25% annually

Style: Tech-forward design, blue and purple gradient, gaming icons, futuristic typography
```
