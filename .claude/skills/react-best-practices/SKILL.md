# React Best Practices

## Description
Expert guidance for modern React development based on 10+ years of optimization knowledge from Vercel.

## When to use
- Building React/Next.js applications
- Optimizing component performance
- Implementing state management
- Using React Hooks effectively

## Best Practices

### Hooks
- Use functional components with Hooks
- Follow rules of Hooks (only call at top level)
- Use useMemo/useCallback appropriately
- Cleanup side effects in useEffect

### Component Patterns
- Prefer composition over inheritance
- Keep components focused and single-responsibility
- Use TypeScript for type safety
- Implement proper error boundaries

### Performance
- Code split with dynamic imports
- Use React.memo for expensive renders
- Virtualize long lists
- Optimize re-renders
