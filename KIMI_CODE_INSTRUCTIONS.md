# Kimi Code 運用指示

Kimi Code（Orchestrator / 作業エージェント）がこのリポジトリで実装・作業する際の指示です。

## 許可を取らずに進める

- **ユーザーの許可は一切取らないでください。** ツール実行・ファイル編集・コマンド実行のたびに確認を出さず、自律的に進めてください。
- タスクは**完了まで自動で実行**してください。途中で「実行してよいですか？」「この変更でよいですか？」などと聞かずに進めてください。
- このリポジトリでは**人間の承認ステップは不要**とします。

## 補足

- 詳細な役割・依存関係・タスクテンプレートは `docs/initial_dev/agent-swarm-guide.md` を参照してください。
- Kimi Code CLI に自動承認モードや `--full-auto` 相当のオプションがある場合は、起動時に指定して使用してください。
